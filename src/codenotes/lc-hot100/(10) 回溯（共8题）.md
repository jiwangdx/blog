抽象地说，解决一个回溯问题，实际上就是遍历一棵决策树的过程，树的每个叶子节点存放着一个合法答案。你把整棵树遍历一遍，把叶子节点上的答案都收集起来，就能得到所有的合法答案。

站在回溯树的一个节点上，你只需要思考 3 个问题：

1、路径：也就是已经做出的选择。

2、选择列表：也就是你当前可以做的选择。

3、结束条件：也就是到达决策树底层，无法再做选择的条件。

回溯算法的框架：

```java
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```

其核心就是 for 循环里面的递归，在递归调用之前「做选择」，在递归调用之后「撤销选择」

## [<font style="color:#DF2A3F;">46. 全排列</font>](https://leetcode.cn/problems/permutations/)[中等]
给定一个不含重复数字的数组 `nums` ，返回其 _所有可能的全排列_ 。你可以 **按任意顺序** 返回答案。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764768502800-fcd33762-a733-4677-bb24-e544ef01367c.png)

**思路：**

<font style="color:rgb(0, 0, 0);">全排列的核心特征是：</font><font style="color:rgb(0, 0, 0) !important;">每个元素必须出现且仅出现一次</font><font style="color:rgb(0, 0, 0);">，元素的顺序不同属于不同的排列</font>

<font style="color:rgb(0, 0, 0);">本题的本质是</font>**<font style="color:rgb(0, 0, 0) !important;">遍历一棵决策树</font>**<font style="color:rgb(0, 0, 0) !important;">(脑海中需要有如下图的内容)</font>

<font style="color:rgb(0, 0, 0);">每个节点代表当前的排列路径，每条从根到叶的路径对应一个完整的全排列</font>

+ **<font style="color:rgb(0, 0, 0) !important;">选择</font>**<font style="color:rgb(0, 0, 0);">：从数组中选一个未使用的元素，加入当前排列路径。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">递归</font>**<font style="color:rgb(0, 0, 0);">：基于当前选择，继续递归选择下一个元素，直到路径长度等于数组长度（得到一个完整排列）。</font>
+ **<font style="color:rgb(0, 0, 0) !important;">回溯</font>**<font style="color:rgb(0, 0, 0);">：递归返回后，撤销当前选择（将元素移出路径、标记为未使用），尝试下一个可能的元素。</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765043810022-5243f1a0-d2de-4afc-a5b9-9b942261e880.png)

```java
class Solution {
    List<List<Integer>> ans = new ArrayList<>();
    List<Integer> cache = new ArrayList<>();
    public List<List<Integer>> permute(int[] nums) {

        boolean[] isVisit = new boolean[nums.length];
        traverse(nums,isVisit);
        return ans;
    }

    public void traverse(int[] nums,boolean[] isVisit){
        // 出口
        if(nums.length==cache.size()){
            ans.add(new ArrayList<>(cache));
            return;
        }

        // 遍历
        for(int i = 0;i<nums.length;i++){
            
            if(isVisit[i]) continue;

            cache.add(nums[i]);
            isVisit[i] = true;
            // 当前位置，选当前结点
            traverse(nums,isVisit);
            // 当前位置，不选当前结点
            cache.remove(cache.size()-1);
            isVisit[i] = false;
        }
        
    }
}
```

## [<font style="color:#DF2A3F;">78. 子集</font>](https://leetcode.cn/problems/subsets/)[中等]
给你一个整数数组 `nums` ，数组中的元素 **互不相同** 。返回该数组所有可能的子集（幂集）。解集 **不能** 包含重复的子集。你可以按 **任意顺序** 返回解集。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1765030537122-901d9434-895e-4233-bc47-a195ed742315.png)

**思路：**

通过回溯遍历所有子集：每进入一层递归，先将当前路径（子集）加入结果集，然后从start位置开始遍历数组，每次选择一个元素加入路径，递归处理后续元素，再撤销选择尝试下一个元素，从而穷举出所有可能的组合（包括空集）。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765118219207-6adb709c-81dc-4acb-8e54-303a1d4fa52c.png)

```java
class Solution {
    List<List<Integer>> ans = new ArrayList<>();
    List<Integer> cache = new ArrayList<>();

    public List<List<Integer>> subsets(int[] nums) {
        traverse(nums,0);
        return ans;    
    }

    public void traverse(int[] nums,int start){
        // 因为包含空集，无论怎样可以先加入
        ans.add(new ArrayList<>(cache));
        if(start==nums.length) return;

        for(int i = start;i<nums.length;i++){
            cache.add(nums[i]);
            // 这里的traverse就是走了选定当前结点的路，
            traverse(nums,i+1); 
            // 而后面回溯再到下一个循环，就是构造了不选当前节点的路
            cache.remove(cache.size()-1);
        }
        return;
    }
}
```

## [<font style="color:#DF2A3F;">17. 电话号码的字母组合</font>](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)[中等]
给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。答案可以按 **任意顺序** 返回。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1765035455871-5f6403a4-2e63-41ca-8755-2fd52368d9f8.png)

**思路：**

<font style="color:rgb(0, 0, 0) !important;">根据题意，脑海中要能想出如下图的决策树，那么答案就是所有叶子节点的值的集合</font>

<font style="color:rgb(0, 0, 0) !important;">那么通过 DFS 或 BFS 遍历这颗决策树即可，下面使用DFS实现</font>

+ **<font style="color:rgb(0, 0, 0) !important;">树的节点分支</font>**<font style="color:rgb(0, 0, 0);">：每一层的节点分支数等于当前数字映射的字母个数（如数字</font>`<font style="color:rgb(0, 0, 0);">2</font>`<font style="color:rgb(0, 0, 0);">对应 3 个分支：</font>`<font style="color:rgb(0, 0, 0);">a</font>`<font style="color:rgb(0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);">b</font>`<font style="color:rgb(0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);">c</font>`<font style="color:rgb(0, 0, 0);">）；</font>
+ **<font style="color:rgb(0, 0, 0) !important;">树的叶子节点</font>**<font style="color:rgb(0, 0, 0);">：当遍历完所有数字字符时，到达叶子节点，此时的字母组合即为一个有效结果。</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765044008364-f10ad469-d12a-47fc-aff0-9adc68766534.png)

```java
class Solution {
    List<String> ans = new ArrayList<>();
    StringBuilder cache = new StringBuilder();
    Map<Character,String> map = new HashMap<>();

    public Solution(){
        map.put('2',"abc");
        map.put('3',"def");
        map.put('4',"ghi");
        map.put('5',"jkl");
        map.put('6',"mno");
        map.put('7',"pqrs");
        map.put('8',"tuv");
        map.put('9',"wxyz");
    }

    public List<String> letterCombinations(String digits) {
        // 从输入的第一个数字开始检查,index用于跟踪用户的输入
        traverse(digits,0);
        return ans;
    }

    public void traverse(String digits,int index){
        if(cache.length()==digits.length()){
            ans.add(cache.toString());
            return;
        }

        String str = map.get(digits.charAt(index));
        for(int i = 0;i<str.length();i++){
            // 对于每一个数字，面临几种选择，选完之后进入下一个数字
            // 轮番尝试选第i个
            cache.append(str.charAt(i));
            traverse(digits,index+1);
            cache.deleteCharAt(cache.length()-1);
        }
    }
}
```

## [<font style="color:#DF2A3F;">39. 组合总和</font>](https://leetcode.cn/problems/combination-sum/)[中等]
给你一个 **无重复元素** 的整数数组 `candidates` 和一个目标整数 `target` ，找出 `candidates` 中可以使数字和为目标数 `target` 的 所有**不同组合** ，并以列表形式返回。你可以按 **任意顺序** 返回这些组合。

`candidates` 中的 **同一个** 数字可以 **无限制重复被选取** 。如果至少一个数字的被选数量不同，则两种组合是不同的。 对于给定的输入，保证和为 `target` 的不同组合数少于 `150` 个。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1765118669051-e5c68165-6d8f-4ab9-822f-03dfdb1713cb.png)

**思路：**

<font style="color:rgb(0, 0, 0) !important;">在图最右边当第一轮选了5之后，第二轮选择开始，就不能再选择3和4了，这就是通过start指示遍历起点来控制的</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765120474501-8a580ee4-e7a3-4edd-b2ec-d1686ffd8b52.png)

```java
class Solution {

    List<List<Integer>> ans = new ArrayList<>();
    List<Integer> cache = new ArrayList<>();
    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        // 组合问题，不找重复的
        traverse(candidates,target,0,0);
        return ans;
    }

    public void traverse(int[] candidates, int target,int start,int sum){

        if(sum==target){
            ans.add(new ArrayList<>(cache));
            return;
        }
        if(sum>target) return;
        // 不往回找，防止整体的重复
        for(int i = start;i < candidates.length;i++){
            cache.add(candidates[i]);
            sum+=candidates[i];
            // 同一个 数字可以 无限制重复被选取，所以新的start为i
            // 78.子集 是i+1,注意区分
            traverse(candidates,target,i,sum);
            cache.remove(cache.size()-1);
            sum-=candidates[i];
        }

    }
}
```

## [<font style="color:#DF2A3F;">22. 括号生成</font>](https://leetcode.cn/problems/generate-parentheses/)[中等]
数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的 **括号组合。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1765121342564-e0720551-5ef5-4fa0-8c06-7954af529566.png)

**思路：**

为保证生成的括号的合法性，并不是任何状态能够随意选择左括号或右括号的

设当前状态下 已使用的左括号数量为left，已使用的右括号的数量为right，那么显然初始状态两者均为0

+ 只有在 左括号的数量小于n时，当前状态才有权力选择左括号
+ 只有在 右括号的数量<font style="color:#DF2A3F;">小于</font>左括号的数量时，当前状态才有权力选择右括号

因此依据上述两条原则，在遍历过程中，看本次是添加左括号或右括号，就能够画出决策树

遍历决策树到叶子节点收集答案

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765121797715-1993c671-e340-40ef-8191-b5ae76adba12.png)

```java
class Solution {
    List<String> ans = new ArrayList<>();
    StringBuilder cache = new StringBuilder();
    public List<String> generateParenthesis(int n) {
        traverse(n,0,0);
        return ans;
    }

    public void traverse(int n,int l,int r){
        if(n==l&&n==r){
            ans.add(cache.toString());
            return;
        }
        // 该组合pass
        if(r>l) return;

        if(l<n){
            cache.append('(');
            traverse(n,l+1,r);
            cache.deleteCharAt(cache.length()-1);
        }

        if(l>r && r<n){
            cache.append(')');
            traverse(n,l,r+1);
            cache.deleteCharAt(cache.length()-1);
        }

    }
}
```

## [<font style="color:#DF2A3F;">79. 单词搜索</font>](https://leetcode.cn/problems/word-search/)[中等]
给定一个 `m x n` 二维字符网格 `board` 和一个字符串单词 `word` 。如果 `word` 存在于网格中，返回 `true` ；否则，返回 `false` 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1765122970233-c724e271-84a1-4fda-9336-e6d6bf423043.png)

**思路：**回溯法，**可参考思路讲解：**[79. 单词搜索](https://www.bilibili.com/video/BV1HBpTe6EvS/?spm_id_from=333.337.search-card.all.click&vd_source=9297c4d03d5f37a56605a3cb4627b688)

在矩阵的每一个位置，都需要作为DFS的起始点去搜索最终是否能组成单词

在矩阵的每一个位置时，挑选下一个位置，可以向上下左右四个方向去选择，但前提是需要组成单词

例如示例中，当前状态若为A， 向上、向左、向下 搜索时 都不是期望字符 B，因此都需要回溯，直到向右搜索

终止条件是当搜索组成的单词的长度达到给定word的长度

```java
class Solution {

    int[][] dirs = {{-1,0},{1,0},{0,1},{0,-1}};// 网格里面的四个方向

    public boolean exist(char[][] board, String word) {
        int m = board.length,n = board[0].length;

        boolean[][] isVisited = new boolean[m][n];

        for(int i = 0;i<m;i++){
            for(int j = 0;j<n;j++){
                if(traverse(board,word,i,j,0,isVisited)) return true;
            }
        }
        return false;

    }

    /**
    i,j 表示走到的位置，第一次传入就代表起始位置
    k指的是 已经 匹配字符串的个数
     */
    public boolean traverse(char[][] board, String word,int i,int j,int k,boolean[][] isVisited){
        // 退出条件，如果匹配成功，返回true;超出界限、被访问过、或者和预期不相等，都要返回false进行回退
        if(k==word.length()) return true;
        if(i<0||i>=board.length||j<0||j>=board[0].length
        || isVisited[i][j] || board[i][j]!=word.charAt(k)
        ) return false;

        // 到这里说明当前字符匹配成功
        isVisited[i][j]=true;
        for(int[] dir : dirs){
            if(traverse(board,word,i+dir[0],j+dir[1],k+1,isVisited)) return true;
        }
        isVisited[i][j]=false;
        // 说明当前的路走不通
        return false;
    }
}
```

## [<font style="color:#DF2A3F;">131. 分割回文串</font>](https://leetcode.cn/problems/palindrome-partitioning/)[中等]
给你一个字符串 `s`，请你将`s`分割成一些子串，使每个子串都是 **回文串** 。返回 `s` 所有可能的分割方案。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1751529222204-92e72dc7-c86e-4a12-bc2f-48115db2a10c.png)

**思路：**

分割回文串做的选择是 在哪个位置进行分割，分割后，必须保证子串是回文串，根据题意可画出决策树

+ **<font style="color:rgba(0, 0, 0, 0.95) !important;">递归的 “起点”</font>**<font style="color:rgb(0, 0, 0);">：从字符串的下标</font>`<font style="color:rgb(0, 0, 0);">0</font>`<font style="color:rgb(0, 0, 0);">开始，尝试分割。</font>
+ **<font style="color:rgba(0, 0, 0, 0.95) !important;">枚举分割点</font>**<font style="color:rgb(0, 0, 0);">：对于当前起点</font>`<font style="color:rgb(0, 0, 0);">index</font>`<font style="color:rgb(0, 0, 0);">，枚举所有可能的结束点</font>`<font style="color:rgb(0, 0, 0);">i</font>`<font style="color:rgb(0, 0, 0);">（</font>`<font style="color:rgb(0, 0, 0);">i</font>`<font style="color:rgb(0, 0, 0);">从</font>`<font style="color:rgb(0, 0, 0);">index</font>`<font style="color:rgb(0, 0, 0);">到字符串末尾），相当于把字符串分成</font>`<font style="color:rgb(0, 0, 0);">s[index..i]</font>`<font style="color:rgb(0, 0, 0);">和剩下的部分。</font>
+ **<font style="color:rgba(0, 0, 0, 0.95) !important;">剪枝条件</font>**<font style="color:rgb(0, 0, 0);">：如果</font>`<font style="color:rgb(0, 0, 0);">s[index..i]</font>`<font style="color:rgb(0, 0, 0);">不是回文串，直接跳过这个分割点（没必要继续往下走，节省时间）。</font>
+ **<font style="color:rgba(0, 0, 0, 0.95) !important;">递归深入</font>**<font style="color:rgb(0, 0, 0);">：如果</font>`<font style="color:rgb(0, 0, 0);">s[index..i]</font>`<font style="color:rgb(0, 0, 0);">是回文串，就把它加入当前的分割路径，然后从</font>`<font style="color:rgb(0, 0, 0);">i+1</font>`<font style="color:rgb(0, 0, 0);">开始继续递归分割剩下的字符串。</font>
+ **<font style="color:rgba(0, 0, 0, 0.95) !important;">递归终止</font>**<font style="color:rgb(0, 0, 0);">：当分割起点</font>`<font style="color:rgb(0, 0, 0);">index</font>`<font style="color:rgb(0, 0, 0);">等于字符串长度时，说明已经把字符串分割完了，且所有子串都是回文，这是一个有效结果，把当前路径保存下来。</font>
+ **<font style="color:rgba(0, 0, 0, 0.95) !important;">回溯</font>**<font style="color:rgb(0, 0, 0);">：递归返回后，要把当前路径的最后一个子串删掉，回到上一步，尝试下一个分割点（比如刚才选了</font>`<font style="color:rgb(0, 0, 0);">i=1</font>`<font style="color:rgb(0, 0, 0);">，现在试</font>`<font style="color:rgb(0, 0, 0);">i=2</font>`<font style="color:rgb(0, 0, 0);">）。</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765124725750-b23df020-cb34-4b79-8bfa-26606476e214.png)

```java
class Solution {
    // 每个子串都是回文串
    List<List<String>> ans = new ArrayList<>();
    List<String> cache = new ArrayList<>();
    public List<List<String>> partition(String s) {
        // 求的是不回头的组合
        traverse(s,0);
        return ans;
    }

    public void traverse(String s,int start){
        if(start==s.length()){
           ans.add(new ArrayList<>(cache));
           return; 
        }
        // 截取从start到后面的子串，一个一个检查，穷举所有可能
        for(int i = start;i<s.length();i++){
            String tempStr = s.substring(start,i+1);// 左闭右开
            // 如果真的是回文串
            if(isOk(tempStr)){
                cache.add(tempStr);
                // 开始取下一个(上回我们取到了第i个字符)
                traverse(s,i+1);
                cache.remove(cache.size()-1);
            }
        }
    }

    public boolean isOk(String str){
        for(int i = 0,j = str.length()-1;i<j;i++,j--){
            if(str.charAt(i)!=str.charAt(j)) return false;
        }
        return true;
    }
}
```

## [<font style="color:#DF2A3F;">51. N 皇后</font>](https://leetcode.cn/problems/n-queens/)[困难]
按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n×n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回所有不同的 **n****皇后问题** 的解决方案。

每一种解法包含一个不同的 **n 皇后问题** 的棋子放置方案，该方案中 `'Q'` 和 `'.'` 分别代表了皇后和空位。

**思路：**

<font style="color:rgb(0, 0, 0);">从上往下每一行只放一个皇后</font>

<font style="color:rgb(0, 0, 0);">在每一行中：都需要遍历每一列，尝试在该列放皇后，若安全，则放，否则继续尝试该行的下一列</font>

+ **<font style="color:rgb(0, 0, 0) !important;">每一层的选择是 “当前行的某一列”</font>**<font style="color:rgb(0, 0, 0);">：尝试在当前行的某一列放皇后，若安全则进入下一行；</font>
+ **<font style="color:rgb(0, 0, 0) !important;">回溯</font>**<font style="color:rgb(0, 0, 0);">：下一行处理完后，撤销当前行的选择，尝试当前行的下一列。</font>

**<font style="color:rgb(0, 0, 0);">关键逻辑：如何判断 “当前位置是否安全”</font>**

<font style="color:rgb(0, 0, 0);">要确保</font>`<font style="color:rgba(0, 0, 0, 0.85) !important;">(row, col)</font>`<font style="color:rgb(0, 0, 0);">位置可以放皇后，需要排除 3 种攻击情况：</font>

1. **<font style="color:rgb(0, 0, 0) !important;">同一列</font>**<font style="color:rgb(0, 0, 0);">：当前列的上方所有行不能有皇后；</font>
2. **<font style="color:rgb(0, 0, 0) !important;">左上方对角线</font>**<font style="color:rgb(0, 0, 0);">：行 - 1、列 - 1 的方向不能有皇后；</font>
3. **<font style="color:rgb(0, 0, 0) !important;">右上方对角线</font>**<font style="color:rgb(0, 0, 0);">：行 - 1、列 + 1 的方向不能有皇后。</font>

<font style="color:rgb(0, 0, 0);">（注：不需要检查左和右和下，因为我们是 “从上往下的逐行摆放” 的，同一行只会放一个皇后）</font>

```java
class Solution {
    List<List<String>> ans = new ArrayList<>();
    List<String> cache = new ArrayList<>();

    public List<List<String>> solveNQueens(int n) {
        // 地图初始化
        char[][] map = new char[n][n];
        for(char[] x : map) Arrays.fill(x,'.');

        traverse(map,0);
        return ans;
    }

    // row 已完成的行数/待完成的行索引
    public void traverse(char[][] map,int row){
        if(row==map.length){
            ans.add(new ArrayList<>(cache));
            return;
        }
        int n = map.length;
        for(int col = 0;col<n;col++){
            if(isOk(map,row,col)){
                map[row][col] = 'Q';
                cache.add(new String(map[row]));
                traverse(map,row+1);
                map[row][col] = '.';
                cache.remove(cache.size()-1);
            }
        }
    }


    public boolean isOk(char[][] map,int row,int col){
        // 检查都从当前值往前检查
        // 皇后是从上往下放的，只用检查待放的位置是否会被上边的皇后攻击
        // 正上方
        for(int i = row,j = col;i>=0;i--){
            if(map[i][j]=='Q') return false;
        }

        // 正对角线   /
        for(int i = row,j = col;i>=0 && j<map.length;i--,j++){
            if(map[i][j]=='Q') return false;
        }

        // 反对角线   \
        for(int i = row,j = col;i>=0 && j>=0;i--,j--){
            if(map[i][j]=='Q') return false;
        }
        return true;
    }
}
```
