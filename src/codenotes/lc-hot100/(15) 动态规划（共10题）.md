## [<font style="color:#DF2A3F;">70. 爬楼梯</font>](https://leetcode.cn/problems/climbing-stairs/)[简单]
假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

![](https://jiwang.cc.cd/PicGo/1765126269324-95532489-3020-479a-8b70-cd2d870b0bc7.png)

**思路：**

+ <font style="color:rgb(0, 0, 0);">n=1 → 1 种</font>
+ <font style="color:rgb(0, 0, 0);">n=2 → 2 种</font>
+ <font style="color:rgb(0, 0, 0);">n=3 → 3 种（刚好是 n=2 的方法数 + n=1 的方法数：2+1=3）</font>
+ <font style="color:rgb(0, 0, 0);">n=4 → 5 种（刚好是 n=3 的方法数 + n=2 的方法数：3+2=5）</font>
+ <font style="color:rgb(0, 0, 0);">n=5 → 8 种（5+3=8）</font>

观察规律  **<font style="color:rgb(0, 0, 0) !important;">爬到 n 阶的总方法数 = 爬到 n-1 阶的方法数 + 爬到 n-2 阶的方法数</font>**<font style="color:rgba(0, 0, 0, 0.85);">。</font>

+ 递推公式：dp[n] = dp[n-1] + dp[n-2]
+ 边界条件：dp[0] = 1，dp[1] = 1

```java
class Solution {
    public int climbStairs(int n) {
        int[] dp = new int[n+1];
        if(n==1 || n==2) return n; 
        dp[1] = 1;
        dp[2] = 2;
        for(int i = 3;i<=n;i++){
            dp[i] = dp[i-1]+dp[i-2];
        }
        return dp[n];
    }
}
```

## [<font style="color:#DF2A3F;">118. 杨辉三角</font>](https://leetcode.cn/problems/pascals-triangle/)[简单]
给定一个非负整数 `_numRows_`_，_生成「杨辉三角」的前 `_numRows_`行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765209884286-f5e1abdc-3a85-4a06-abdd-45f78066e41e.png)

**思路：**

先将杨辉三角变成下图这种形式，观察规律：

+ 每一行的第一个元素和最后一个元素都是固定为1
+ 每一行除了第一个元素和最后一个元素之外的其他元素=其上方元素+其左上方元素

```java
class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> c = new ArrayList<>(numRows); // 预分配空间
        c.add(List.of(1));
        for (int i = 1; i < numRows; i++) {
            List<Integer> row = new ArrayList<>(i + 1); // 预分配空间
            row.add(1);
            for (int j = 1; j < i; j++) {
                // 左上方的数 + 正上方的数
                row.add(c.get(i - 1).get(j - 1) + c.get(i - 1).get(j));
            }
            row.add(1);
            c.add(row);
        }
        return c;
    }
}
```

## [<font style="color:#DF2A3F;">198. 打家劫舍</font>](https://leetcode.cn/problems/house-robber/)[中等]
你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，**如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警**。

给定一个代表每个房屋存放金额的非负整数数组，计算你** 不触动警报装置的情况下 **，一夜之内能够偷窃到的最高金额。

![](https://jiwang.cc.cd/PicGo/1765210877504-3806e121-ab45-41c5-94bd-fd7845c246cf.png)

**思路：**

以[1、2、3、1]为例，由于相邻的两间房不能同一晚偷窃

+ 假设不偷窃最后一间房，那么偷盗 [1、2、3、1] 四间房屋的最大利润就变成了偷盗 [1、2、3] 三间房屋能获得的最大利润
+ 假设偷窃最后一间房(那么倒数第二间就不能偷了)，那么偷盗 [1、2、3、1] 四间房屋的最大利润就变成了偷盗 [1、2] 两间房屋的最大利润 + 偷盗最后一间房屋所能获得的金额（即1） 

偷盗[1、2、3]或者偷盗[1、2]都可以按上述递归下去求解，直到边界条件 :

+ 偷盗 0 间房的最大利润为 0
+ 偷盗第1间房的最大利润为num[0]

因此总结归纳规律，偷盗n间房的最大利润dp[n] = Math.max( `偷盗n-1间房的最大利润dp[n-1]`，`偷盗n-2间房的最大利润dp[n-2]+nums[n-1]`)，前者代表没有偷盗第n间房，后者代表偷盗了第n间房

+ dp[i] 表示偷窃前 i 个房屋的最大金额
+ 例如`<font style="color:rgb(0, 0, 0);">dp[4] = max(dp[3]=4, dp[2]+nums[3]=2+1=3)</font>`<font style="color:rgba(0, 0, 0, 0.85);"> → </font>`<font style="color:rgb(0, 0, 0);">dp[4]=4</font>`<font style="color:rgba(0, 0, 0, 0.85);">。</font>

```java
class Solution {
    public int rob(int[] nums) {
        int n = nums.length;
        // 保存子问题的解, dp[i] 表示偷窃前i个房屋的最大金额
        int[] dp = new int[n + 1];
        dp[0] = 0;          // 前0个房屋，利润为0
        dp[1] = nums[0];    // 前1个房屋，利润为nums[0]
        for (int i = 2; i <= n; i++) {
            // 两种选择：
            // 1. 不偷第i个房屋，利润为dp[i-1]（前i-1个房屋的最大利润）
            // 2. 偷第i个房屋，利润为dp[i-2] + nums[i-1]（前i-2个房屋的最大利润 + 第i个房屋的金额，nums[i-1]对应第i个房屋）
            dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i-1]);
        }
        return dp[n];
    }
}
```

## [<font style="color:#DF2A3F;">322. 零钱兑换</font>](https://leetcode.cn/problems/coin-change/)[中等]
给你一个整数数组 `coins` ，表示不同面额的硬币；以及一个整数 `amount` ，表示总金额。

计算并返回可以凑成总金额所需的 **最少的硬币个数** 。如果没有任何一种硬币组合能组成总金额，返回 `-1` 。

你可以认为每种硬币的数量是无限的。

![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1765213482080-d17e3607-91a0-472e-8659-c876ae549116.png)

**思路：**

```java
class Solution {
    public int coinChange(int[] coins, int amount) {

        int[] dp = new int[amount + 1];
        Arrays.fill(dp, amount + 1);

        dp[0] = 0;
        // 这里不能推出dp[1] = 1，∵不一定有1元的硬币

        // 构造金额
        for(int i = 1; i <= amount; i++){

            // 尝试所有硬币
            for(int coin : coins){
                if(i >= coin){
                    dp[i] = Math.min(dp[i], dp[i - coin] + 1);
                }
            }

        }

        return dp[amount] == amount + 1 ? -1 : dp[amount];
    }
}
```

## [<font style="color:#DF2A3F;">279. 完全平方数</font>](https://leetcode.cn/problems/perfect-squares/)[中等]
给你一个整数 `n` ，返回 _和为 _`_n_`_ 的完全平方数的最少数量_ 。

**完全平方数** 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，`1`、`4`、`9` 和 `16` 都是完全平方数，而 `3` 和 `11` 不是。

![](https://jiwang.cc.cd/PicGo/1765298091377-9a932186-4695-4b05-95c6-466f80d84bf9.png)

**思路：**

```java
class Solution {
    public int numSquares(int n) {
        // 完全背包，背包容量是n,完全平方数就是不同价值的物品，求最小
        int[] dp = new int[n+1];
        // dp[i]代表和为i的完全平方数的最少数量
        // 最小的完全平方数是1，数n最多只能由n个构成
        Arrays.fill(dp,n+1);

        dp[0] = 0;
        dp[1] = 1;
        for(int i = 2;i<=n;i++){ //构造容量

            for(int j = 1;j*j<=i;j++){ //遍历平方数
                dp[i] = Math.min(dp[i],dp[i-j*j]+1);
            }

        }
        return dp[n];
        // 这里不需要像这样写，因为一定有解
        // return dp[n]==n+1? -1:dp[n];
    }
}
```



## [<font style="color:#DF2A3F;">139. 单词拆分</font>](https://leetcode.cn/problems/word-break/)[中等]
给你一个字符串 `s` 和一个字符串列表 `wordDict` 作为字典。如果可以利用字典中出现的一个或多个单词拼接出 `s` 则返回 `true`。

**注意：**不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。

![](https://jiwang.cc.cd/PicGo/1765299573504-d424ab69-8ff0-47d8-9539-b31ca95436ef.png)

**思路：**

```java
class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {

        int n = s.length();

        boolean[] dp = new boolean[n + 1];

        // 空字符串
        dp[0] = true;

        // 构造字符串前缀
        for(int i = 1; i <= n; i++){

            // 枚举所有单词
            for(String word : wordDict){

                int len = word.length();

                if(i >= len && dp[i-len] && s.substring(i-len, i).equals(word)){
                    dp[i] = true;
                    break;
                }
            }
        }

        return dp[n];
    }
}
```

## [<font style="color:#DF2A3F;">300. 最长递增子序列</font>](https://leetcode.cn/problems/longest-increasing-subsequence/)[中等]（序列DP）
给你一个整数数组 `nums` ，找到其中最长严格递增子序列的长度。

**子序列 **是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，`[3,6,2,7]` 是数组 `[0,3,1,6,2,2,7]` 的子序列。

![](https://jiwang.cc.cd/PicGo/1765302167472-01f91dc0-607c-4d97-bee3-5efa22e9097b.png)

**思路：**

dp[n] = 以第n个元素结尾的最长子序列。 

```java
class Solution {
    public int lengthOfLIS(int[] nums) {

        int n = nums.length;

        // dp[i] 表示：以 nums[i] 结尾的最长递增子序列长度
        int[] dp = new int[n];

        // 每个元素至少可以单独成为一个长度为1的序列
        Arrays.fill(dp, 1);

        int ans = 1;

        // 从第二个元素开始，逐个计算以 nums[i] 结尾的 LIS
        for (int i = 1; i < n; i++) {

            // 枚举 i 之前的所有元素，尝试接在它们后面
            for (int j = 0; j < i; j++) {

                // 如果 nums[i] 可以接在 nums[j] 后面，形成更长的递增序列
                if (nums[j] < nums[i]) {

                    // 更新以 i 结尾的最长序列长度
                    dp[i] = Math.max(dp[i], dp[j] + 1);
                }
            }

            // 记录全局最大递增子序列长度
            ans = Math.max(ans, dp[i]);
        }

        return ans;
    }
}
```

## [<font style="color:#DF2A3F;">152. 乘积最大子数组</font>](https://leetcode.cn/problems/maximum-product-subarray/)[中等]
给你一个整数数组 `nums` ，请你找出数组中乘积最大的非空连续 子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 **32位** 整数。

![](https://jiwang.cc.cd/PicGo/1765302252307-15269c28-81d9-418e-9edb-fa23de030452.png)

**思路：**

```java
class Solution {
    public int maxProduct(int[] nums) {
        // 整体遍历取最大
        int preMax = nums[0];
        int preMin = nums[0];
        int ans = nums[0]; //先将其赋值为第一个元素的值，保证每个值都考虑到
        
        for(int i = 1;i<nums.length;i++){
            int curMax = Math.max(Math.max(preMax*nums[i],preMin*nums[i]),nums[i]);
            int curMin = Math.min(Math.min(preMax*nums[i],preMin*nums[i]),nums[i]);
            preMax = curMax;
            preMin = curMin;
            ans = Math.max(curMax,ans);
        }
        return ans;
    }
}
```

## [<font style="color:#DF2A3F;">416. 分割等和子集</font>](https://leetcode.cn/problems/partition-equal-subset-sum/)[中等]
给你一个 **只包含正整数 **的 **非空 **数组 `nums` 。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

![](https://jiwang.cc.cd/PicGo/1765302298132-d13671da-2424-4be4-9462-3f2d8b95d3f7.png)

**思路：**

nums所有元素之和为偶数，才有可能使得数组分割成两个子集且两个子集的元素和相等

若nums所有元素之和为奇数，则不可能达成目的

因此问题可以转化为可否在数组中找到子集，其子集元素和=数组所有元素和的一半

dp[i] ：和为i的子集是否存在

这本质上是0-1背包，固定采用这种先遍历物品，然后逆序，防止同一物品被重复使用

```java
class Solution {
    public boolean canPartition(int[] nums) {
        int n = nums.length;
        int total = 0;
        for(int num : nums){
            total+=num;
        }
        if(total%2!=0) return false;
        int target = total/2;

        boolean[] dp = new boolean[target+1];
        dp[0] = true;
        for(int num:nums){
            for(int i = target;i>=num;i--){
                if(dp[i-num]) dp[i] = true;
                if(dp[target]) return true;
            }
        }
        return false;
    }
}
```

## [<font style="color:#DF2A3F;">32. 最长有效括号</font>](https://leetcode.cn/problems/longest-valid-parentheses/)[困难]
给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

![](https://jiwang.cc.cd/PicGo/1765302318014-1db1af9e-dcad-420c-a227-1ff26b2637ab.png)

**思路：**

```java
// 有效括号的两个必要条件：
// 数量平衡：左右括号数量相等

// 顺序合法：在任何前缀中，左括号数量 ≥ 右括号数量（从左到右看）

class Solution {
    public int longestValidParentheses(String s) {
        int n = s.length();
        int ans = 0;
        
        // 从左到右扫描
        int l = 0, r = 0;
        for (int i = 0; i < n; i++) {
            char c = s.charAt(i);
            if (c == '(') {
                l++;
            } else {
                r++;
            }
            if (l == r) {
                ans = Math.max(ans, 2 * l);
            } else if (r > l) { 
                // 右括号多于左括号，说明前面的部分不合法,下个循环从头开始计算左右括号数量
                l = r = 0;
            }
        }
        
        // 从右到左扫描
        l = 0; r = 0;
        for (int i = n - 1; i >= 0; i--) {
            char c = s.charAt(i);
            if (c == ')') {
                r++;
            } else {
                l++;
            }
            if (l == r) {
                ans = Math.max(ans, 2 * l);
            } else if (l > r) { // 左括号多于右括号，说明后面的部分不合法
                l = r = 0;
            }
        }
        
        return ans;
    }
}
```
