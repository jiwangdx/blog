## [<font style="color:#DF2A3F;">73. 矩阵置零</font>](https://leetcode.cn/problems/set-matrix-zeroes/)[中等]
给定一个 `_m_`` x ``_n_` 的矩阵，如果一个元素为 **0 **，则将其所在行和列的所有元素都设为 **0** 。请使用 [**原地**](http://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95) 算法**。**

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1762959159740-55a34838-9478-4002-b21d-c29a563d2448.png)

**思路：**

**使用第一行和第一列分别用来标识某一列和某一行是否存在0**

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        // 先看第一行和第一列有没有0，把结果记录下来
        boolean firstRow = false;
        boolean firstCol = false;
        // 第一行有没有0
        for(int i = 0;i<n;i++){
            if(matrix[0][i]==0) {
                firstRow=true; 
                break;
            }
            
        }
        // 第一列有没有0
        for(int i = 0;i<m;i++){
            if(matrix[i][0]==0) {
                firstCol=true; 
                break;
            }
        }

        // 注意后边两部分就是从1开始了

        for(int i = 1;i<m;i++){
            for(int j = 1;j<n;j++){
                if(matrix[i][j]==0){
                    matrix[0][j] = 0; // 第j列为0
                    matrix[i][0] = 0; // 第i行为0
                }
            }
        }

        for(int i = 1;i<m;i++){
            for(int j = 1;j<n;j++){
                if(matrix[0][j] == 0||matrix[i][0] == 0){
                    matrix[i][j]=0;
                }
            }
        }

        if(firstRow){ //第一行有0，那第一行就全置为0
            for(int i = 0;i<n;i++){
                matrix[0][i] = 0;
            }
        }
        if(firstCol){ //第一列有0，那第一列就全置为0
            for(int i = 0;i<m;i++){
                matrix[i][0] = 0;
            }
        }
    }
}
```

## [<font style="color:#DF2A3F;">54. 螺旋矩阵</font>](https://leetcode.cn/problems/spiral-matrix/)[中等]
给你一个 `m` 行 `n` 列的矩阵 `matrix` ，请按照 **顺时针螺旋顺序** ，返回矩阵中的所有元素。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1762960688397-dfd0f50a-8077-4006-84fe-429c7e2334ad.png)

**思路：**

通过动态维护四个边界（top、bottom、left、right）来模拟顺时针收缩的遍历过程，按照“向右→向下→向左→向上”的顺序依次遍历矩阵外层，每遍历完一条边就更新对应的边界，当边界交错时终止循环。

这里边界处理采用“左闭右闭”。

```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;

        List<Integer> ans = new ArrayList<>();
        // 指定上下左右边界
        int top = 0;
        int bottom = m-1;
        int left = 0;
        int right = n-1;

        while(true){
            for(int i = left;i<=right;i++){
                ans.add(matrix[top][i]);
            }
            if(++top>bottom) break;

            for(int i = top;i<=bottom;i++){
                ans.add(matrix[i][right]);
            }
            if(--right<left) break;

            for(int i = right;i>=left;i--){
                ans.add(matrix[bottom][i]);
            }
            if(--bottom<top) break;

            for(int i = bottom;i>=top;i--){
                ans.add(matrix[i][left]);
            }
            if(++left>right) break;

        }
        return ans;
    }
}
```

## [<font style="color:#DF2A3F;">48. 旋转图像</font>](https://leetcode.cn/problems/rotate-image/)[中等]
给定一个 _n _× _n_ 的二维矩阵 `matrix` 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在[**原地**](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95) 旋转图像，这意味着你需要直接修改输入的二维矩阵。**请不要 **使用另一个矩阵来旋转图像。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1762963295758-ec52cc20-0cef-4baf-a6fb-dd131d62e90a.png)

**思路：**

先转置再每行反转

```java
class Solution {
    public void rotate(int[][] matrix) {
        // 先转置，再swap
        int m = matrix.length;
        int n = matrix[0].length;  
        for(int i = 0;i<m;i++){
            for(int j = i;j<n;j++){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }

        for(int i = 0;i<m;i++){
            swap(matrix[i]);
        }

    }

    public void swap(int[] arr){
        int n = arr.length;
        for(int i = 0;i<n/2;i++){
            int temp = arr[i];
            arr[i] = arr[n-i-1];
            arr[n-i-1] = temp;
        }
    }
}
```

## [<font style="color:#DF2A3F;">240. 搜索二维矩阵 II</font>](https://leetcode.cn/problems/search-a-2d-matrix-ii/)[中等]
编写一个高效的算法来搜索 `_m_`` x ``_n_` 矩阵 `matrix` 中的一个目标值 `target` 。该矩阵具有以下特性：

+ 每行的元素从左到右升序排列。
+ 每列的元素从上到下升序排列。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1762966124314-dd8680e6-b7b1-4a80-bd07-d6292dc10faf.png)

**思路：**从右上角开始与目标值比较

若右上角的数 大于 目标值，那么这一列的数都会大于目标值，那么可以排除这一整列，即 j--

若右上角的数 小于 目标值，那么这一行的数都会小于目标值，那么可以排除这一整行，即 i++

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int i = 0;
        int j = matrix[0].length - 1; // 从右上角开始
        while (i < matrix.length && j >= 0) { // 还有剩余元素
            if (matrix[i][j] == target) {
                return true; // 找到 target
            }
            if (matrix[i][j] > target) {
                j--;  // 这一列的数都会大于目标值，排除
            } else {
                i++;  // 这一行的数都会小于目标值，排除
            }
        }
        return false;
    }
}
```
