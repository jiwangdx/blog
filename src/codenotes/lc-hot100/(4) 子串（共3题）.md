## [<font style="color:#DF2A3F;">560. 和为 K 的子数组</font>](https://leetcode.cn/problems/subarray-sum-equals-k/)[中等]
给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 _该数组中和为 _`_k_`_的子数组的个数 _。

子数组是数组中元素的连续非空序列。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774277968-44fd45.png)

**思路：**

+ **前缀和+哈希表：**遍历数组时计算前缀和，将前缀和作为key，出现的次数作为value存进哈希表。对于遍历到下标`i`时，只需要知道在之前的前缀和值为`preSum[i] - target`出现的次数即可，然后把次数相加

```java
// 核心思路：子数组的和可以表示为 两个前缀和的差
class Solution {
    public int subarraySum(int[] nums, int k) {
        int count = 0;
        int preSum = 0; 	// 不断变化的前缀和
        HashMap<Integer, Integer> mp = new HashMap<>(); // 记录前缀和以及前缀和出现的次数
        
        // 含义为：初始前缀和为 0 出现了 1 次 
        // 或者理解为可能存在preSum[i] == k的情况，为这种情况做铺垫
        mp.put(0, 1);	
        
        for (int i = 0; i < nums.length; i++) {
            
            preSum += nums[i];
            // 核心逻辑: 对于每个新的前缀和 preSum值,检查preSum-k是否存在Map中
            if (mp.containsKey(preSum - k)) {
                count += mp.get(preSum - k);
            }
            // 若当前的 前缀和存在则value+1, 若不存在则插入该前缀和为key & 置value为1
            mp.put(preSum, mp.getOrDefault(preSum, 0) + 1); 
        }
        return count;
    }
}
```

## [<font style="color:#DF2A3F;">239. 滑动窗口最大值</font>](https://leetcode.cn/problems/sliding-window-maximum/)[困难]
给你一个整数数组 `nums`，有一个大小为 `k`的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。返回 _滑动窗口中的最大值 _。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774277968-d408c6.png)

**思路：**

<font style="color:rgb(15, 17, 21);">可以把双端队列想象成一个"候选人队列"，它有以下特点：</font>

1. **<font style="color:rgb(15, 17, 21);">单调递减</font>**<font style="color:rgb(15, 17, 21);">：队首的值最大，往队尾方向值逐渐变小</font>
2. **<font style="color:rgb(15, 17, 21);">时效性</font>**<font style="color:rgb(15, 17, 21);">：只存储当前窗口内和可能成为未来最大值的元素</font>
3. **<font style="color:rgb(15, 17, 21);">淘汰机制</font>**<font style="color:rgb(15, 17, 21);">：</font>
    - <font style="color:rgb(15, 17, 21);">过期的淘汰（超出窗口范围）</font>
    - <font style="color:rgb(15, 17, 21);">弱小的从队尾淘汰（比新来的小）</font>

**<font style="color:rgb(15, 17, 21);">工作流程：</font>**

+ <font style="color:rgb(15, 17, 21);">每次新元素进来时，先把过期的候选人踢掉</font>
+ <font style="color:rgb(15, 17, 21);">再把比自己小的候选人都踢掉（因为它们永远不可能成为最大值了）</font>
+ <font style="color:rgb(15, 17, 21);">然后自己加入候选人队列</font>
+ <font style="color:rgb(15, 17, 21);">当前窗口的最大值就是队首的候选人</font>

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        // 滑动窗口内最大值
        // 把最大值放在最前面

        // 维护单调队列
        // 清理过期从队头出，维护单调从队尾出，新元素从队尾进，最大值从队头取。
        
        Deque<Integer> window = new ArrayDeque<>();
        
        // <============>
        int n = nums.length;

        int[] ans = new int[n-k+1];
        int w = 0;// 跟踪数组的插入

        for(int i = 0;i<n;i++){


            // 如果队头过期（保证窗口大小）
            if(!window.isEmpty() && i-k+1>window.peekFirst()) window.pollFirst();

            // 维护最大值
            // 把右边小于自己的都给移出去
            while(!window.isEmpty() && nums[window.peekLast()]<nums[i]) window.pollLast();   

            // 压进去索引就好
            window.offerLast(i);// 从尾部压进去

            // 读一下
            if(i>=k-1){
                ans[w] = nums[window.peekFirst()];
                w++;
            }
            
        
        }

        return ans;
             
    }
}
```

## [<font style="color:#DF2A3F;">76. 最小覆盖子串</font>](https://leetcode.cn/problems/minimum-window-substring/)[困难]
给你一个字符串 `s` 、一个字符串 `t` 。返回 `s` 中涵盖 `t` 所有字符的最小子串。如果 `s` 中不存在涵盖 `t` 所有字符的子串，则返回空字符串 `""` 。

**注意：**

+ 对于 `t` 中重复字符，我们寻找的子字符串中该字符数量必须不少于 `t` 中该字符数量。
+ 如果 `s` 中存在这样的子串，我们保证它是唯一的答案。

**什么是「涵盖」**

<font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);">看示例 1，</font>_<font style="color:rgba(38, 38, 38, 0.75);background-color:rgb(240, 240, 240);">s</font>_<font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);"> 的子串 </font><font style="color:rgba(38, 38, 38, 0.75);background-color:rgb(240, 240, 240);">BANC</font><font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);"> 中每个字母的出现次数，都</font>**<font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);">大于等于</font>**<font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);"> </font>_<font style="color:rgba(38, 38, 38, 0.75);background-color:rgb(240, 240, 240);">t</font>_<font style="color:rgba(38, 38, 38, 0.75);background-color:rgb(240, 240, 240);">=ABC</font><font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);"> 中每个字母的出现次数，这就叫</font>**<font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);">涵盖</font>**<font style="color:rgb(38, 38, 38);background-color:rgb(240, 240, 240);">。</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774277968-bedc42.png)

**思路：**

**注意，不是一次就能定位右端点，左端点移动到不符合要求之后，右端点还要一直遍历直到末尾**

```java
class Solution {
    public String minWindow(String s, String t) {
        if (s.length() < t.length()) return "";
        
        // need: t中字符的需求计数
        int[] need = new int[128];
        // window: 当前窗口中字符的计数
        int[] window = new int[128];
        
        // 统计t中字符需求
        for (char c : t.toCharArray()) {
            need[c]++;
        }
        
        // 需要满足的字符种类数（need中计数>0的字符数）
        int needCount = 0;
        for (int count : need) {
            if (count > 0) needCount++;
        }
        
        int left = 0, right = 0;
        int valid = 0;  // 当前已满足需求的字符种类数

        int start = 0, minLen = Integer.MAX_VALUE;
        
        while (right < s.length()) {
            // 扩大右边界
            char c = s.charAt(right);
            right++;
            
            // 如果是t中需要的字符
            if (need[c] > 0) {
                window[c]++;
                // 如果该字符的数量刚好满足需求
                if (window[c] == need[c]) {
                    valid++;
                }
            }
            
            // 当窗口满足所有需求时
            while (valid == needCount) {
                // 更新最小窗口
                if (right - left < minLen) {
                    minLen = right - left;
                    start = left;
                }
                
                // 收缩左边界
                char d = s.charAt(left);
                left++;
                
                // 如果这是我们需要的数
                if (need[d] > 0) {
                    // 如果现在的数量处于我们需要的临界点
                    if (window[d] == need[d]) {
                        valid--;
                    }
                    // 不属于就单纯把它的量减去1
                    window[d]--;
                }
            }
            // 跳出内部小循环的时候又变成了valid小于need
        }
        
        return minLen == Integer.MAX_VALUE ? "" : s.substring(start, start + minLen);
    }
}
```
