## [<font style="color:#DF2A3F;">3. 无重复字符的最长子串</font>](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)[中等]
**题目：**给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串 **的长度。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1752986345376-acddbb0a-bf9e-40eb-bfc5-8d811a830630.png)

**思路：**

由两个索引维护一个区间，满足一定条件时会动态的去调整这个区间。

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        // 字符串的比较
        if(s.equals("")) return 0;

        // 有一个窗口一直用来存无重复字符的子串
        // 整个队列吧
        int n = s.length();
        Deque<Character> window = new ArrayDeque<>();

        int ans = 1;
        for(int i = 0;i<n;i++){
            
            while(window.contains(s.charAt(i))){
                // poll()方法默认是队头出列
                window.poll();
            };
            window.offer(s.charAt(i));

            // 这时窗口里边都是满足要求的
            ans = Math.max(ans,window.size());      
        }
        return ans;
    }
}
```

## [<font style="color:#DF2A3F;">438. 找到字符串中所有字母异位词</font>](https://leetcode.cn/problems/find-all-anagrams-in-a-string/)[中等]
**题目：**给定两个字符串 `s` 和 `p`，找到 `s`中所有 `p`的 **异位词 **的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1752986369834-5f196be6-39df-49bb-97ea-40a1ee9dde82.png)<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1759589713702-f529e66f-e8fb-4090-90b5-12585c900047.png)

**思路：**

因为字符串`p`的长度是固定的，那么就需要在字符串`s`中找到长度相同的子串

即需要遍历字符串`s`中所有长度与p相同的子串，此时就可以使用滑动窗口来解决

把当前窗口组成的子串排序，以及字符串p内部也排序，如果两个字符串排序后相同，则说明当前窗口的子串就是字符串p的异位词。

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        String flag = tool(p);// 找到判断异位词的标杆

        int len_s = s.length();
        int len_p = p.length(); 
        List<Integer> ans = new ArrayList<>();
        for(int i = 0;i<=len_s-len_p;i++){
            String temp = s.substring(i,i+len_p);
            if(tool(temp).equals(flag)) ans.add(i) ;
        }
        return ans;    
    }



    public String tool(String str){
        char[] chs = str.toCharArray();
        Arrays.sort(chs);
        return new String(chs);
    } 
}
```
