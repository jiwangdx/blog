## [<font style="color:#DF2A3F;">1. 两数之和</font>](https://leetcode.cn/problems/two-sum/)[简单]
给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值 **`target` 的那 **两个** 整数，并**返回它们的数组下标**。

你可以假设每种输入只会对应一个答案，并且你<font style="color:#DF2A3F;">不能使用两次相同的元素</font>。你可以按任意顺序返回答案。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774277713-1fa169.png)

**思路：**

使用哈希表存储， key存储数组值、value存储下标

遍历数组 nums，i 为当前下标，每个值都判断map中是否存在 key==target-nums[i] 的值

如果存在，则找到了两个值，如果不存在，则将当前的 (nums[i] , i) 存入 map 中，继续遍历直到找到为止

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int n = nums.length;
        Map<Integer,Integer> map = new HashMap<>();
        for(int i = 0;i<n;i++){
            if(map.containsKey(target-nums[i])){
                return new int[]{i,map.get(target-nums[i])};
            }
            map.put(nums[i],i);
        } 
        return new int[0];
    }
}
```

## [<font style="color:#DF2A3F;">49. 字母异位词分组</font>](https://leetcode.cn/problems/group-anagrams/)[中等]
给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词** 是通过重新排列不同单词或短语的字母而形成的单词或短语，并使用所有原字母一次<font style="color:rgb(38, 38, 38);">。</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774277713-c8818a.png)

**思路：**

把同一类词进行<font style="color:#DF2A3F;">分组</font>，使用Map（key: 同一类词的相同唯一特征, value: 字母异位词的集合）。

```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        int n = strs.length;
        List<List<String>> ans = new ArrayList<>();
        Map<String,List<String>> map = new HashMap<>();
        for(int i = 0;i<n;i++){
            String str = strs[i];
            String target = strSort(str);
            if(!map.containsKey(target)){
                List<String> cache = new ArrayList<>();
                cache.add(str);
                map.put(target,cache);
            }
            else{
                map.get(target).add(str);
            }

        }

        return new ArrayList<>(map.values());

    }

    public String strSort(String str){
        char[] chs = str.toCharArray();
        Arrays.sort(chs);
        return new String(chs);
    }
}
```

## [<font style="color:#DF2A3F;">128. 最长连续序列</font>](https://leetcode.cn/problems/longest-consecutive-sequence/)[中等]
给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。请你设计并实现时间复杂度为 `O(n)`的算法解决此问题。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774277713-44ccbe.png)

**思路：**

+ 先将所有数字存入哈希表，用于快速判断某个数字是否存在；
+ 然后遍历<font style="color:#DF2A3F;">哈希表</font>中的每个数字num：
    - 如果 num - 1 存在，说明这个数字num不是某个连续序列的起点，跳过；
    - 如果 num - 1 不存在，则从当前数字num开始向后查找，统计连续递增序列的长度；
+ 记录并更新最大长度。

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        if(nums==null||nums.length==0) return 0;
        // 需要去重，都放进hashset里面，然后遍历hashset
        int n = nums.length;

        Set<Integer> set = new HashSet<>();
        for(int i =0;i<n;i++){
            set.add(nums[i]);
        }
        int ans = 1;
        for(int x:set){
            int k = 1;
            if(set.contains(x-1)) continue;
            while(set.contains(x+1)){
                k++;
                x++;
            }
            ans = Math.max(ans,k);
        }

        return ans;
    }
}
```
