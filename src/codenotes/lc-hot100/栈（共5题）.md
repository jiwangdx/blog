## [<font style="color:#DF2A3F;">20. 有效的括号</font>](https://leetcode.cn/problems/valid-parentheses/)[简单]
**题目：**给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1759653569820-8c475047-91d1-4fe2-b9c5-ecc19b215be4.png)

**思路：**

遍历字符串的每个字符，如果遇到左括号就放入栈中，如果遇到右括号，就取出栈顶的左括号与之匹配

若不匹配说明当前字符串不是有效的括号，直接返回false，若匹配，则消掉栈顶的左括号和当前右括号

当遍历完成后，若栈内是空的，则说明当前字符串是有效的括号，若栈内还剩余有左括号，说明不符合

```java
class Solution {
    public boolean isValid(String s) {
        Deque<Character> stack = new ArrayDeque<>();
        // 使用数组作为映射，ASCII码作为索引
        char[] map = new char[128];
        map[')'] = '(';
        map[']'] = '[';
        map['}'] = '{';
        
        for (char c : s.toCharArray()) {
            // 如果是右括号
            if (map[c] != 0) {
                // 如果可以搞（栈非空且栈匹配成功）
                if (!stack.isEmpty() && stack.peek() == map[c]) {
                    stack.pop();
                }
                else return false;
            } else {
                // 左括号入栈
                stack.push(c);
            }
        }
        
        return stack.isEmpty();
    }
}
```

## [<font style="color:#DF2A3F;">155. 最小栈</font>](https://leetcode.cn/problems/min-stack/)[中等]
**题目：**设计一个支持 `push` ，`pop` ，`top` 操作，并能在<font style="color:#DF2A3F;">常数时间内检索到最小元素的栈</font>。

实现 `MinStack` 类:

+ `MinStack()` 初始化堆栈对象。
+ `void push(int val)` 将元素val推入堆栈。
+ `void pop()` 删除堆栈顶部的元素。
+ `int top()` 获取堆栈顶部的元素。
+ `int getMin()` 获取堆栈中的最小元素。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1759657089954-5f2986f4-0e1d-4497-bd17-378420e55e03.png)

**思路：**

+ 需要两个栈，主栈用于保存正常push进的元素，一个辅助栈 min_stack，用于存 以及 获取 stack 中最小值
+ **算法流程：**
    - push() 方法： 每当 push() 新值进来时，如果`新值 <= min_stack的栈顶值`，则把新值也存入 `min_stack`，使得 min_stack 的栈顶始终是整个栈 stack 中的最小值。
    - pop() 方法： 若`当前要弹出的元素== min_stack 的栈顶值`，那么也要把min_stack的栈顶值弹出，这样可以保证 min_stack 栈顶元素始终是整个栈 stack 中的最小值
    - getMin()方法： 返回 min_stack 栈顶元素即可
    - top()方法：返回 stack 栈顶元素即可
+ **min_stack 作用分析：**
    - 因为 stack 是可能弹出元素的，所以整个栈的最小值是动态变化的，有可能 最小值由 a 变成 b（因为弹出b） 再变成 a ；因此最小值也需要动态的记录，而 **<font style="color:rgb(15, 17, 21);">min_stack 专门用于跟踪和存储主栈中的最小值序列</font>**<font style="color:rgb(15, 17, 21);">，它维护了一个"最小值的历史记录"，使得我们能够在O(1)时间内获取当前栈中的最小元素。</font>
+ `<font style="color:rgb(15, 17, 21);">x <= min_stack.peek()</font>`<font style="color:rgb(15, 17, 21);">   </font><font style="color:#DF2A3F;">为什么是等于呢？===> 最小值可能不止一个</font><font style="color:rgb(15, 17, 21);">。</font>

```java
class MinStack {
    private Deque<Integer> stack;
    private Deque<Integer> min_stack;

    public MinStack() {
        stack = new ArrayDeque<>();
        min_stack = new ArrayDeque<>();
    }
    public void push(int x) {
        stack.push(x);
        // 新值 <= min_stack的栈顶值，则把新值也存入 min_stack
        if (min_stack.isEmpty() || x <= min_stack.peek()) {
            min_stack.push(x);
        }
    }
    public void pop() {
        int stackTop = stack.peek();
        stack.pop();
        // 若当前要弹出的元素是最小值
        if (stackTop==min_stack.peek()) {
            min_stack.pop();
        }
    }
    public int top() {
        return stack.peek();
    }
    public int getMin() {
        return min_stack.peek();
    }
}
```

## [<font style="color:#DF2A3F;">394. 字符串解码</font>](https://leetcode.cn/problems/decode-string/)[中等]
**题目：**给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为: `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。注意 `k` 保证为正整数。

+ 你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。
+ 此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1759661479904-6871d817-33ed-4d22-bb75-125537604c38.png)

**思路：**

需要借助一个栈，遍历字符串的每个字符，遍历过程中

1. 只要字符不是右括号']'，就将字符压入栈，直到遇到右括号']'
2. 遇到右括号']'，就不用入栈了，此时只要栈顶的元素不是左括号'['，就把栈顶元素弹出，即<font style="color:#DF2A3F;">解析最里层[]内的字符串</font>，用temp暂存。
3. 遇到了左括号'['，说明最里层 [] 内的字符串已经解析完毕，即temp
4. 弹出栈顶的左括号'['
5. 接着解析左括号左边的数字,可能是多位,也可能是0，<font style="color:#DF2A3F;">弹栈直到栈顶不为数字</font>
6. 当数字解析完毕后，根据 []内的字符串 和 [] 左边的数字 重新拼接一个字符串
7. 拼接完成，把新的字符串压入栈中

遍历过程完成后，将stack里的结果取出拼接后返回即可

+ 如果是像这种有多个括号分开的"3[a]2[bc]",最终stack中会有两个字符串[aaa,bcbc]因此需要将栈里的字符串拼接起来
+ 如果是像有一个整体大括号包裹的"3[a2[c]]",那么最终stack中就只会有一个字符串[accaccacc]

```java
class Solution {
    // s 中所有整数的取值范围为 [1, 300] 

    public String decodeString(String s) {
        Deque<String> stack = new ArrayDeque<>();
        
        for (char ch : s.toCharArray()) {
            if (ch != ']') {
                stack.push(String.valueOf(ch));
            } else {
                String str = "";
                while(!stack.peek().equals(String.valueOf('['))){
                    str = stack.pop() + str;
                }
                stack.pop(); // 把最后一个左括号弹掉

                String num = "";
                // 函数积累1：判断字符是不是数字
                while(!stack.isEmpty() && Character.isDigit(stack.peek().charAt(0))){
                    num = stack.pop() + num;
                }
                // 函数积累2：字符串转整数
                int repeatNum = Integer.valueOf(num);
                String newStr = "";
                for(int i = 0; i < repeatNum; i++){
                    newStr += str;
                }
                // 如果字符串中有嵌套的括号结构，比如 "3[a2[c]]"，那么内层的结果需要作为外层字符串的一部分继续参与计算。
                stack.push(newStr);
            }
        }
        String res = "";
        while(!stack.isEmpty()){
            // 函数积累3：字符串拼接
            res = stack.pop() + res;
        }
        return res;
    }
}
```

## [<font style="color:#DF2A3F;">739. 每日温度</font>](https://leetcode.cn/problems/daily-temperatures/)[中等]
**题目：**给定一个整数数组 `temperatures` ，表示每天的温度，返回一个数组 `answer` ，其中 `answer[i]` 是指对于第 `i` 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 `0` 来代替。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1759672135474-de99a233-885b-468f-b6bc-c3a06b9f0cf2.png)

**思路：**

我们使用 单调递减栈 来存储数组下标，栈中的下标对应的温度是递减的（<font style="color:#DF2A3F;">从栈底到栈顶</font>）

遍历每日温度数组，当我们遍历到一个新的温度时

+ <font style="color:#DF2A3F;">如果</font>`<font style="color:#DF2A3F;">当前遍历的温度>栈顶对应的温度</font>`<font style="color:#DF2A3F;">，那么就找到了栈顶元素的下一个更高温度</font>，计算(当前遍历的温度和栈顶温度的)天数差，且将天数差存入对应位置，最后将栈顶元素弹出栈顶（因为它的下一个更高温度已经找到了，没有用处了）
+ 如果 当前遍历的温度 小于等于 栈顶对应的温度，那么我们就将当前温度的索引压入栈，保持栈的单调性

当遍历完成，如果栈中还有元素，那么说明 其对应的温度 是没有下一个更高温度的，其结果默认为 0，符合要求

```java
class Solution {
    public int[] dailyTemperatures(int[] temperatures) {
        // 栈存储的是 温度数组的下标索引，而非值
        Deque<Integer> stack = new ArrayDeque<>();  
        
        int[] res = new int[temperatures.length];
        // 遍历每日温度数组
        for(int i = 0; i < temperatures.length; i++){
            // 若栈不为空,且 今天的温度 大于 栈顶温度，
            // 那么就找到了 距离栈顶温度最近的 下一个更高温度: 即今天
            while(!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]){
                // 栈顶温度那天的下标
                int index = stack.peek();   
                // 距离栈顶温度最近的下一个更高温度出现在 = [今天-栈顶温度那天的下标] 天后
                res[index] = i - index;  
                // 已经找到了栈顶温度那天的结果，无用了就弹出
                stack.pop();                
            }
            
            // 今天的温度的下标先存入栈
            stack.push(i);                 
        }
        return res;
    }
}
```

## [<font style="color:#DF2A3F;">84. 柱状图中最大的矩形</font>](https://leetcode.cn/problems/largest-rectangle-in-histogram/)[困难]
**题目：**给定 _n_ 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

求在该柱状图中，能够勾勒出来的矩形的最大面积。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1759672164700-1907c655-e137-4ffe-910a-3848e00ac92e.png)             <!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1759672174501-456e56b7-8811-466b-b9bc-109b104e211f.png)

**思路：**对每一个柱子，需要获取到它<font style="color:#DF2A3F;">左右两边的第一个小于等于该柱子高度的柱子的位置</font>，`以当前柱子为高度`，`左右两边第一个小于该高度的柱子之间的距离作为宽度`,求出当前柱子能勾勒出的矩形的面积，记录下最大值。

例如 以示例一中的 `柱子 5` 为例，左边第一个小于 5 的是 `柱子 1 (索引为1，记为left)`，右边第一个小于 5 的是 `柱子2 (索引为4，记为right)`，因此 柱子5 所能勾勒出的矩形的面积是 `高(5) x 宽(right - left -1 = 2) = 10`

那么对每一个柱子，如何获取到柱子左右两边第一个小于当前高度的柱子的位置呢？我们可以使用 left[] 和 right[] 两个数组 进行记录，规定：

+ left[i] 表示 柱子[i] 左边第一个小于 柱子[i] 高度的柱子的索引位置，例如 柱子5 对应的是 left[2] = 1
+ right[i]  表示 柱子[i] 右边第一个小于 柱子[i] 高度的柱子的索引位置，例如 柱子5 对应的是 right[2] = 4

<font style="color:#DF2A3F;">结合单调递增栈，遍历数组一遍，即可把 left[] 和 right[] 都填满</font>

```java
class Solution {
    public int largestRectangleArea(int[] heights) {
        // 小于的时候收获，维护一个单调递增栈
        // 我需要知道左边第一个小于的，和右边第一个小于的
        Deque<Integer> stack = new ArrayDeque<>();
        int n = heights.length;
        int[] left = new int[n];
        int[] right = new int[n];
        // 初始化left和right数组
        for(int i = 0; i < n; i++){
            left[i] = -1;
            right[i] = n;
        }

        for(int i = 0;i<n;i++){
            // 1 2 4 5             3   
            // 严格小于    
            while(!stack.isEmpty() && heights[i]<heights[stack.peek()]){
                right[stack.peek()] = i;
                stack.pop();
            }
            if(!stack.isEmpty()){
                left[i] = stack.peek();
            }
            stack.push(i);
        }

        int ans = Integer.MIN_VALUE;
        for(int i = 0;i<n;i++){
            ans = Math.max(ans,(right[i]-left[i]-1)*heights[i]);
        }

        return ans;

    }
}
```

<font style="color:#DF2A3F;">怎么判断是用单调栈类型？</font>看什么时候收获。题目中是找更小的数，那就应该用递增栈，因为递增栈是遇到小的触发收获。每日温度是要找大的，那就用递减栈，遇到大的就干活。
