## [<font style="color:#DF2A3F;">160. 相交链表</font>](https://leetcode.cn/problems/intersection-of-two-linked-lists/)[简单]
给你两个单链表的头节点 `headA` 和 `headB` ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null` 。
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-d7d938.png)

**思路：**指针p1遍历完链表A后接着遍历链表B，而指针p2遍历完链表B后接着遍历链表A

如此那么两个指针就一定会同时遍历到两个链表的相交的起始节点。

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        if(headA==null||headB==null) return null;
        // 互相走彼此的路，最终会相遇
        ListNode p1 = headA;
        ListNode p2 = headB;
        while(p1!=p2){
            if(p1!=null) p1 = p1.next;
            else p1 = headB;
            if(p2!=null) p2 = p2.next;
            else p2 = headA;     
        }
        return p1;
    }
}
```

## [<font style="color:#DF2A3F;">206. 反转链表</font>](https://leetcode.cn/problems/reverse-linked-list/)[简单]
给你单链表的头节点 `head` ，请你反转链表，并返回反转后的链表。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-af79d4.png)

**思路**：定义 pre 指针(指向前一个元素)和 cur 指针(指向当前元素)，遍历链表，在遍历的过程中不断的把 链表节点之间的 指针进行反转，即 cur.next = pre。接着 pre 和 cur 指针都向后移动，接着重复上述过程。直到最后，cur 会移动到 null，而 pre 会移动到最后一个节点。此时就可以返回了，因此，while 循环的条件应该是 cur != null，且应当返回 pre 指针。这个题看动画或结合图才会比较清楚。

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode cur = head;
        ListNode pre = null;
        while(cur!=null){
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur  = temp; 
        }
        return pre;
    }
}

```

## [<font style="color:#DF2A3F;">234. 回文链表</font>](https://leetcode.cn/problems/palindrome-linked-list/)[简单]
给你一个单链表的头节点 `head` ，请你判断该链表是否为回文链表。如果是，返回 `true` ；否则，返回 `false` 。
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-e90c0a.png)

**思路：**

**方案一**：将链表的内容复制到数组，再使用数组双指针一头一尾遍历判断是否回文

```java
class Solution {
    public boolean isPalindrome(ListNode head) {
        ArrayList<Integer> num = new ArrayList<>();
        // 将链表的内容复制到数组
        ListNode p = head;
        while (p != null) {
            num.add(p.val);
            p = p.next;
        }
        // 数组双指针一头一尾遍历判断是否回文
        for (int left = 0, right = num.size() - 1; left < right; left++, right--) {
            if (num.get(left) != num.get(right)) {
                return false;
            }
        }
        return true;
    }
}
```

**方案二(优)**：寻找中间节点+反转后半段的链表。再用链表双指针一头一尾都向中间走，遍历判断是否回文

```java
class Solution {
    public boolean isPalindrome(ListNode head) {
        if (head == null)
            return true;
        //               	 head       head2
        // 1->2->3->2->1      1->2->3<-2<-1   其中3的next为null  
        ListNode midNode = findMidNode(head); // 找到中间结点
        ListNode temp = midNode.next;
        midNode.next = null;
        ListNode head2 = reverseList(temp); // 反转后半部分链表
        while (head2 != null) { // 注意反转后，head2部分的链表最后一个节点是指向null的
            // 一头一尾同时向中间走
            if (head.val != head2.val) {
                return false;
            } else {
                head = head.next;
                head2 = head2.next;
            }
        }
        return true;
    }

    // 876.链表的中间结点：快慢指针,慢指针每次走一步，快指针每次走两步，最终slow就是中间节点
    // 最后找到的中间位 偏左和偏右 ，取决于起始位置
    // 本题为偏左，如果想偏右，fast的起始位置也为head;
    public ListNode findMidNode(ListNode head) {
        ListNode slow = head;
        ListNode fast = head.next;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
        }
        return slow;
    }

    // 206.反转链表
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode cur = head;
        while (cur != null) {
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = temp;
        }
        return pre;
    }
}
```

## [<font style="color:#DF2A3F;">141. 环形链表</font>](https://leetcode.cn/problems/linked-list-cycle/)[简单]
给你一个链表的头节点 `head` ，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 `pos` 来表示链表尾连接到链表中的位置（索引从 0 开始）。**注意：**pos 不作为参数进行传递 。仅仅是为了标识链表的实际情况。

_如果链表中存在环_ ，则返回 `true` 。 否则，返回 `false` 。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-08041d.png)

**思路：**快慢指针，初始时快慢指针都指向head头节点，慢指针每次走一步，快指针每次走两步，**如果链表中存在环**，那么快指针一定会追上慢指针(套圈)，也就是说两个指针一定会相遇。但**如果快指针或快指针的next指针指向空则说明链表中不存在环**。

```java
class Solution {
    public boolean hasCycle(ListNode head) { 
        // 必须同一起跑线
        ListNode fast = head;
        ListNode slow = head;
        while(fast != null && fast.next != null){
            fast = fast.next.next;
            slow = slow.next;
            if(fast == slow){
                return true;	// 相遇,存在环
            } 
        }
        return false;
    }
}
```

## [<font style="color:#DF2A3F;">142. 环形链表 II</font>](https://leetcode.cn/problems/linked-list-cycle-ii/)[中等]
给定一个链表的头节点 `head` ，返回链表开始入环的第一个节点。 _如果链表无环，则返回 _`_null_`_。_

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 `pos` 来表示链表尾连接到链表中的位置（**索引从 0 开始**）。如果 `pos` 是 `-1`，则在该链表中没有环。**注意：**`**pos**`** 不作为参数进行传递**，仅仅是为了标识链表的实际情况。**不允许修改 **链表。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-8c5476.png)

**思路：**

当 slow 和 fast 相遇时，将 slow 或 fast 任意选一个指针 重新指向head

然后 slow 和 fast 同时每次走一步，最终就会在 链表的 环的第一个节点相遇

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-436e42.png)

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        // 有环的话，快慢指针会相遇
        // 开始fancy
        ListNode slow = head;
        ListNode fast = head;

        while(fast != null && fast.next != null){
            fast = fast.next.next;
            slow = slow.next;
            if(fast==slow) break;
        }

        if(fast == null || fast.next == null){
            return null;
        }

        // 这时候两个恶霸相遇，强行掰开，重走长征路
        fast = head;
        while(fast!=slow){
            fast = fast.next;
            slow = slow.next;
        }

        return fast;
    }
}
```

## [<font style="color:#DF2A3F;">21. 合并两个有序链表</font>](https://leetcode.cn/problems/merge-two-sorted-lists/)[简单]
将两个升序链表合并为一个新的**升序**链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-1bb07d.png)

**思路：**当 l1 和 l2 都不是空链表时，判断 l1 和 l2 哪一个链表的头节点的值更小，将较小值的节点添加到结果里，并且将对应链表中的节点向后移一位。不断循环。

我们设定一个哨兵节点 dummy，这可以在最后让我们比较容易地返回合并后的链表。

在循环终止的时候， l1 和 l2 至多有一个是非空的。由于输入的两个链表都是有序的，所以不管哪个链表是非空的，它包含的所有元素都比前面已经合并链表中的所有元素都要大。这意味着我们只需要简单地将非空链表接在合并链表的后面，并返回合并链表即可。

```java
class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        ListNode dummy = new ListNode();
        ListNode p = dummy;
        ListNode p1 = list1;
        ListNode p2 = list2;
        while(p1 != null && p2 != null){
            if(p1.val > p2.val){
                p.next = p2;
                p2 = p2.next;
            }else{
                p.next = p1;
                p1 = p1.next;
            }
            p = p.next;
        }
        if(p1 != null){
            p.next = p1;
        }else{
            p.next = p2;
        }
        return dummy.next;
    }
}
```

## [<font style="color:#DF2A3F;">2. 两数相加</font>](https://leetcode.cn/problems/add-two-numbers/)[中等]
给你两个**非空**的链表，表示两个非负的整数。它们每位数字都是按照**逆序**的方式存储的，并且每个节点只能存储**一位**数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-1d33be.png)

思路：

+ <font style="color:rgb(15, 17, 21);">342 逆序存储为：2 → 4 → 3</font>
+ <font style="color:rgb(15, 17, 21);">465 逆序存储为：5 → 6 → 4</font>
+ <font style="color:rgb(15, 17, 21);">342+465=807  逆序存储为：7 → 0 → 8</font>

<font style="color:rgb(15, 17, 21);">题目里逆序存储数字在链表中，其实会让我们的计算更方便！因为两个数的加法都是从个位数开始相加的</font>

<font style="color:rgb(15, 17, 21);">因此，只需要顺序遍历两条链表的节点，进行相加，注意保留好进位即可</font>

```java
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        if(l1 == null && l2 == null){
            return null;
        }
        ListNode p1 = l1;
        ListNode p2 = l2;
        int carry = 0;						// 保存进位
        ListNode dum = new ListNode(-1);    // 虚拟头节点
        ListNode p = dum;
        // 只要链表还有数或者有进位就得进入循环
        while(p1 != null || p2 != null || carry != 0){  
            int sum = carry;
            if(p1 != null){
                sum += p1.val;
                p1 = p1.next;
            }
            if(p2 != null){
                sum += p2.val;
                p2 = p2.next;
            }
            carry = sum / 10;   // 进位(给下个循环使用)
            int temp = sum % 10; // 当前位的值
            
            // 新建节点,衔接到结果链表后
            ListNode tempNode = new ListNode(temp);
            p.next = tempNode;
            p = p.next;
        }
        return dum.next;
    }
}
```

## [<font style="color:#DF2A3F;">19. 删除链表的倒数第 N 个结点</font>](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/)[中等]
给你一个链表，删除链表的倒数第 `n`个结点，并且返回链表的头结点。
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-bf8994.png)

**思路：**要删除倒数第n个结点，只需要找到倒数第n+1个节点

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode dummy  = new ListNode(-1);
        dummy.next = head;
        ListNode p1 = dummy;
        ListNode p2 = dummy;
        for(int i = 0;i<n+1;i++){
            p1 = p1.next;
        } 

        while(p1!=null){
            p1 = p1.next;
            p2 = p2.next;
        }

        p2.next = p2.next.next;
        return dummy.next;
    }
}
```

## [<font style="color:#DF2A3F;">24. 两两交换链表中的节点</font>](https://leetcode.cn/problems/swap-nodes-in-pairs/)[中等]
给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-21872d.png)

**思路：**

**方案一：递归**，符合递归的特性，注意每一分组翻转后，需要将递归结果连接到前一组节点后面

1、从头到尾 进行 递深 (递归深入的过程)

2、会在尾部开始交换（即遇到了递归的终止条件后，需要开始交换了）

3、把交换后的节点返回给上一部分  即归的过程

```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        if(head==null||head.next==null) return head;

        ListNode left = head;
        ListNode right = head.next;

        left.next = swapPairs(right.next);
        right.next = left;

        return right;
    }
}
```

**方案二：迭代**

```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        if(head==null||head.next==null) return head;
        // 迭代
        ListNode dummy = new ListNode(-1);
        dummy.next = head;
        ListNode pre = dummy;
        // 镜头转到第一组
        ListNode cur = dummy.next;
        while(cur!=null&&cur.next!=null){
            ListNode second = cur.next;
            ListNode next = second.next;
            // pre cur second next
            second.next = cur;
            cur.next = next;
            pre.next = second;

            // 此时cur已经被交换到了当前组的第二个，而second已经成了第一个
            // 因此这里pre应该指向cur
            pre = cur;
            cur = next;
        }
        return dummy.next;
    }
}
```

## [<font style="color:#DF2A3F;">25. K 个一组翻转链表</font>](https://leetcode.cn/problems/reverse-nodes-in-k-group/)[困难]
给你链表的头节点 `head` ，每 `k`个节点一组进行翻转，请你返回修改后的链表。

`k` 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 `k`的整数倍，那么请将最后剩余的节点保持原有顺序。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-57068a.png)

**思路：**

**方案一：递归**

+ 递归的终止条件是 当前区间不满足k个节点，所以不需要反转，就直接返回那个区间的头节点，那么实际上需要去数一遍当前区间是否满足k个
+ 递归每一层级的任务是反转当前区间的链表
+ 注意分组翻转后，当前区间需要连接下一个反转区间返回的头节点

```java
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        ListNode tail = head;
        for (int i = 1; i <= k; i++) {
            if (tail == null) {
                return head;
            }
            tail = tail.next;
        }
        // 这时候实际找到的是下一组的开头

        ListNode pre = null;
        ListNode cur = head;
        while (cur != tail) {
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = temp;
        }

        // 注意此时head变成了尾巴;
        head.next = reverseKGroup(tail, k); 
        return pre;
    }
}
```

**方案二：迭代**

```java
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        // 头结点统一操作
        ListNode dummy =  new ListNode(-1);

        dummy.next = head;
        ListNode tail = dummy;
        ListNode pre  = dummy;

        ListNode start = dummy.next; // 标记初始位置
        while(true){
            // 循环去处理,除非......
            for(int i = 0;i<k;i++){
                tail = tail.next;
                if(tail == null) return dummy.next; // 直接返回  
            }

            // 记录下一个开始位置
            ListNode nextStart = tail.next;
            // 断链
            tail.next = null;

            // 处理,并接上前面
            pre.next = reverse(start);
            start.next = nextStart;

            // 更新指针
            pre = start;
            tail = start;
            start = pre.next;

        }
        
    }

    public ListNode reverse(ListNode head){
        // 链表反转直到链表尾
        ListNode pre = null;
        ListNode cur = head;
        while(cur!=null){
            ListNode temp  = cur.next;
            cur.next = pre;
            pre = cur;
            cur = temp;   
        }
        return pre;
    }
}
```

## [<font style="color:#DF2A3F;">138. 随机链表的复制</font>](https://leetcode.cn/problems/copy-list-with-random-pointer/)[中等]
给你一个长度为 `n` 的链表，每个节点包含一个额外增加的随机指针 `random` ，该指针可以指向链表中的任何节点或空节点。

构造这个链表的 [**深拷贝**](https://baike.baidu.com/item/%E6%B7%B1%E6%8B%B7%E8%B4%9D/22785317?fr=aladdin)。 深拷贝应该正好由 `n` 个 **全新** 节点组成，其中每个新节点的值都设为其对应的原节点的值。新节点的 `next` 指针和 `random` 指针也都应指向复制链表中的新节点，并使原链表和复制链表中的这些指针能够表示相同的链表状态。**复制链表中的指针都不应指向原链表中的节点 **。

例如，如果原链表中有 `X` 和 `Y` 两个节点，其中 `X.random --> Y` 。那么在复制链表中对应的两个节点 `x` 和 `y` ，同样有 `x.random --> y` 。

返回复制链表的头节点。

用一个由 `n` 个节点组成的链表来表示输入/输出中的链表。每个节点用一个 `[val, random_index]` 表示：

+ `val`：一个表示 `Node.val` 的整数。
+ `random_index`：随机指针指向的节点索引（范围从 `0` 到 `n-1`）；如果不指向任何节点，则为 `null` 。

你的代码 **只** 接受原链表的头节点 `head` 作为传入参数。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-24a178.png)

**思路：**两次遍历老链表 + 使用哈希表保存老节点与新节点的关系 <老节点，copy的新节点>

第一次遍历老链表，先Copy老节点的值生成新节点，并且保存在Map中，key为老节点，value为新节点

第二次遍历老链表，从Map中取出对应的新节点，并且设置新节点的 next 指针 和 random 指针

```java
class Solution {
    public Node copyRandomList(Node head) {
        Map<Node,Node> map = new HashMap<>();
        Node p = head;
        while(p!=null){
            map.put(p,new Node(p.val));
            p = p.next;
        }

        p = head;
        while(p!=null){
            Node temp = map.get(p);
            temp.next = map.get(p.next);
            temp.random = map.get(p.random);
            p = p.next;
        }

        return map.get(head);
    }
}
```

## [<font style="color:#DF2A3F;">148. 排序链表</font>](https://leetcode.cn/problems/sort-list/)[中等]
给你链表的头结点 `head` ，请将其按 **升序** 排列并返回 **排序后的链表** 。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-60842f.png)

**思路：**递归+归并排序

```java
class Solution {
    public ListNode sortList(ListNode head) {
        
        if(head==null || head.next==null) return head;
        // 没有头结点，这样找中点，中点偏左
        ListNode fast = head.next,slow = head;

        while(fast!=null && fast.next!=null){
            slow = slow.next;
            fast = fast.next.next;
        }

        ListNode nextList = slow.next;
        slow.next = null;

        return merge(sortList(head),sortList(nextList));
    }

    public ListNode merge(ListNode head1,ListNode head2){
        ListNode dummy = new ListNode(-1);
        ListNode head = dummy;

        while(head1!=null && head2!=null){
            if(head1.val<=head2.val){
                head.next = head1;
                head1 = head1.next;
            }
            else{
                head.next = head2;
                head2 = head2.next;
            }
            head = head.next;
        }

        if(head1!=null){
            head.next = head1;
        }
        if(head2!=null){
            head.next = head2;
        }

        return dummy.next;

    }
}
```

## [<font style="color:#DF2A3F;">23. 合并 K 个升序链表</font>](https://leetcode.cn/problems/merge-k-sorted-lists/)[困难]
给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-186635.png)

**思路：**

使用小顶堆，一开始 将所有链表的头节点加到小顶堆中，利用小顶堆的特性：最小的元素会排列在顶部，若执行弹出操作时元素从顶部弹出。因此就可以构造合并出新的升序链表。

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        if(lists.length==0||lists==null) return null;


        PriorityQueue<ListNode> cache = new PriorityQueue<>(lists.length,(a,b)->a.val-b.val);

        ListNode dummy = new ListNode(-1);
        ListNode p = dummy;

        for(int i = 0;i<lists.length;i++){
            if(lists[i]!=null){
                cache.add(lists[i]);
            }
                
        }

        while(!cache.isEmpty()){
            ListNode temp = cache.poll();
            p.next = temp;
            p = p.next;
            if(temp.next!=null){
                cache.add(temp.next);
            }
        }

        return dummy.next;
        
    }
}
```

## [<font style="color:#DF2A3F;">146. LRU 缓存</font>](https://leetcode.cn/problems/lru-cache/)[困难]
请你设计并实现一个满足 [LRU (最近最少使用) 缓存](https://baike.baidu.com/item/LRU) 约束的数据结构。

实现 `LRUCache` 类：

+ `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
+ `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1` 。
+ `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value` ；如果不存在，则向缓存中插入该组 `key-value` 。如果插入操作导致关键字数量超过 `capacity` ，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 必须以 `O(1)` 的平均时间复杂度运行。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/lchot100/2026-1774278236-8035b9.png)

**思路：**使用哈希表和双向链表实现

+ 函数 get 和 put 必须以O(1)的平均时间复杂度运行，可以使用哈希表
+ LRU 算法需要淘汰最久未使用的 key，因此可以使用双向链表: **维护访问顺序，实现O(1)时间复杂度的插入和删除**。如果使用了某个key，先把该key从链表中删除，然后把该key重新插入到链表头部，那么一直没有被使用的key就会逐渐跑到链表尾部，此时如果要淘汰key，只需要从链表尾部删除元素即可

**<font style="color:rgb(15, 17, 21);">get操作：</font>**

1. **<font style="color:rgb(15, 17, 21);">哈希表查找</font>**<font style="color:rgb(15, 17, 21);">：O(1)找到节点</font>
2. **<font style="color:rgb(15, 17, 21);">链表调整</font>**<font style="color:rgb(15, 17, 21);">：将节点移到头部（先删除再头部插入）O(1)</font>

**<font style="color:rgb(15, 17, 21);">put操作：</font>**

+ **<font style="color:rgb(15, 17, 21);">key存在</font>**<font style="color:rgb(15, 17, 21);">：更新值，将节点移到头部（先删除再头部插入）O(1)</font>
+ **<font style="color:rgb(15, 17, 21);">key不存在</font>**<font style="color:rgb(15, 17, 21);">：</font>
    1. <font style="color:rgb(15, 17, 21);">如果容量满：删除尾部节点 O(1)</font>
    2. <font style="color:rgb(15, 17, 21);">创建新节点插入头部 O(1)</font>
    3. <font style="color:rgb(15, 17, 21);">哈希表添加映射 O(1)</font>

```java
class LRUCache {
    // 用map来维护key到的映射（key,value）
    // 使用双向链表的形式，有需要就把结点摘下来或者移动到最前面，最前面默认就是最近访问的元素

// 双链表结点
    class Node{
        int key,val;
        Node pre,next;
        Node(){}
        Node(int key,int val){
            this.key = key;
            this.val = val;
        }
    }

    private int size;
    private int capacity;
    private Node head,tail;

    private Map<Integer,Node> cache = new HashMap<>();

    // LRUCache的构造函数，初始构造肯定有个双向链表，然后还得有个空Map
    public LRUCache(int capacity) {
        this.capacity = capacity;
        this.size = 0;
        // size初始化为0

        head = new Node();
        tail = new Node();
        head.next = tail;
        tail.pre = head;
    }
    
    public int get(int key) {
        if(cache.containsKey(key)){
            removeThis(cache.get(key));
            addFirst(cache.get(key));
            return cache.get(key).val;
        }
        else{
            return -1;
        }
    }
    
    public void put(int key, int value) {
        if(cache.containsKey(key)){
            // 把之前的摘下来，再放最前面
            // 改一下值
            Node tempNode = cache.get(key);
            tempNode.val = value;
            // 动链表
            removeThis(tempNode);
            addFirst(tempNode);
        }
        else{
            Node node = new Node(key,value);
            cache.put(key,node);
            // 刚访问，放在最前面
            addFirst(node);
            size++;
            // 如果超过限制，从后边移除
            if(size>capacity){
                removeLast();
                size--;
            }
            
        }
        
    }

    public void addFirst(Node node){
        node.next = head.next;
        node.pre = head;
        head.next.pre = node;
        head.next = node;    
    }

    public void removeLast(){
        cache.remove(tail.pre.key);
        tail.pre = tail.pre.pre;
        tail.pre.next = tail;
        
    }

    public void removeThis(Node node){
        node.pre.next = node.next;
        node.next.pre = node.pre;
    }

}

/**
 * Your LRUCache object will be instantiated and called as such:
 * LRUCache obj = new LRUCache(capacity);
 * int param_1 = obj.get(key);
 * obj.put(key,value);
 */
```
