## [<font style="color:#DF2A3F;">94. 二叉树的中序遍历</font>](https://leetcode.cn/problems/binary-tree-inorder-traversal/)[简单]
**题目：**给定一个二叉树的根节点 `root` ，返回 _它的 __**中序**__ 遍历_ 。

递归遍历(中序遍历是左根右)

```java
class Solution {
    List<Integer> ans = new ArrayList<>();  // 声明为成员变量
    
    public List<Integer> inorderTraversal(TreeNode root) {
        dfs(root);
        return ans;
    }
    
    private void dfs(TreeNode node) {
        if (node == null) {
            return;
        }
        dfs(node.left);   // 左
        ans.add(node.val); // 根
        dfs(node.right);  // 右
    }
}
```

迭代写法思路(中序遍历是左根右)：使用栈来模拟递归的调用过程，栈记录了我们访问过的路径

1. 首先不断向左深入，把所有左节点压入栈中

2. 弹出栈顶节点（这是当前最左的节点），访问它

3. 然后转向该节点的右子树，重复上述过程

```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<Integer>();
        Deque<TreeNode> stk = new ArrayDeque<TreeNode>();
        //第一个循环是为了处理初始状态
        while (root != null || !stk.isEmpty()) {
        //入栈向左一直走
            while (root != null) {
                stk.push(root);
                root = root.left;
            }
            // 出栈访问右子树
            root = stk.pop();
            res.add(root.val);
            root = root.right;
        }
        return res;
    }
}
```

## [<font style="color:#DF2A3F;">144. 二叉树的前序遍历</font>](https://leetcode.cn/problems/binary-tree-preorder-traversal/)[简单]
递归遍历(前序遍历是根左右)

```java
class Solution {
    List<Integer> ans = new ArrayList<>();  // 声明为成员变量
    
    public List<Integer> preorderTraversal(TreeNode root) {
        dfs(root);
        return ans;
    }
    
    private void dfs(TreeNode node) {
        if (node == null) {
            return;
        }
        ans.add(node.val); // 根
        dfs(node.left);   // 左
        dfs(node.right);  // 右
    }
}
```

迭代遍历思路(前序遍历是根左右)：利用栈模拟递归，第一步，先把根节点压入栈

而后循环处理：1. 弹出栈顶节点（当前要处理的节点）

2. 访问当前节点（前序遍历先访问根节点）

3. 将右子节点压入栈中

4. 将左子节点压入栈中

注意：先右后左，因为栈是后进先出，我们希望左子节点先被处理，所以要后压入栈

```java
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<Integer>();
        if (root == null) {
            return res;
        }
        
        Deque<TreeNode> stack = new LinkedList<>();
        TreeNode node = root;
        while (!stack.isEmpty() || node != null) {
            while (node != null) {
                // 入栈前直接访问
                res.add(node.val);
                stack.push(node);
                node = node.left;
            }
            node = stack.pop();
            node = node.right;
        }
        return res;
    }
}
```

## [<font style="color:#DF2A3F;">145. 二叉树的后序遍历</font>](https://leetcode.cn/problems/binary-tree-postorder-traversal/)[简单]

```java
class Solution {

        ans.add(node.val); // 根
        dfs(node.right);  // 右
    }
}
```

迭代遍历思路(后序遍历是**左右根**)：

采用双端队列，更加优雅，借用头插法实现后序

这里实现的是根右左

```java
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        Deque<Integer> result = new ArrayDeque<>();
        Stack<TreeNode> stack = new Stack<>();
        while (root != null || !stack.isEmpty()) {
            if (root != null) {
                stack.push(root);
                result.offerFirst(root.val); // 出错返回null
                root = root.right;
            } else {
                root = stack.pop();
                root = root.left;
            }
        }

        return new ArrayList(result);
    }
}
```

## [<font style="color:#DF2A3F;">104. 二叉树的最大深度</font>](https://leetcode.cn/problems/maximum-depth-of-binary-tree/)[简单]
**题目：**给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1763911783959-ee1f6255-77d1-4872-8925-68d44703bd98.png)

**思路：**递归，明确子问题和原问题的关系（已知子问题如何求解出原问题）

计算最大深度，可以转化为求  Math.max(左子树的最大深度, 右子树的最大深度)+1

不断递归，直到遇到空节点，返回空节点的最大深度0

```java
class Solution {
    // 分解
    public int maxDepth(TreeNode root) {

        if(root==null) return 0;
        int l = maxDepth(root.left);
        int r = maxDepth(root.right);

        return Math.max(l,r)+1;

    }
}
```

## [<font style="color:#DF2A3F;">226. 翻转二叉树</font>](https://leetcode.cn/problems/invert-binary-tree/)[简单]
**题目：**给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1763912586954-ab62641b-e528-4ac2-a87d-eaf2568e2da6.png)

**思路：**递归，明确子问题和原问题的关系（已知子问题如何求解出原问题）

翻转这棵二叉树，可以转化为 对左子树进行翻转和对右子树进行翻转，再交换左右子树的根节点即可

不断递归，直到遇到空节点才返回

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        // 左右子树进行翻转，然后进行连接
        if(root==null) return null;
        // 不能省事，毕竟要互相调用
        TreeNode lNode = invertTree(root.left);
        TreeNode rNode = invertTree(root.right);
        root.left = rNode;
        root.right = lNode;
        return root;
    }
}
```



## [<font style="color:#DF2A3F;">101. 对称二叉树</font>](https://leetcode.cn/problems/symmetric-tree/)[简单]
**题目：**给你一个二叉树的根节点 `root` ， 检查它是否轴对称。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764172449158-983a111c-db66-4df3-bc16-dde5e47a3c05.png)

**思路：**检查这棵树是否对称，即检查根节点的左右子树是否镜像对称

比较两个树是否是镜像对称：

+ left.val 等于 right.val。
+ left 的左子树与 right 的右子树互为镜像。这是一个和原问题相似的子问题，可以递归判断。
+ left 的右子树与 right 的左子树互为镜像。这是一个和原问题相似的子问题，可以递归判断。

```java
class Solution {
    public boolean isSymmetric(TreeNode root) {
        return root == null || recur(root.left, root.right);
    }
    
    boolean recur(TreeNode L, TreeNode R) {
        if (L == null && R == null) return true;
        if (L == null || R == null || L.val != R.val) return false;
        return recur(L.left, R.right) && recur(L.right, R.left);
    }
}
```

## [<font style="color:#DF2A3F;">543. 二叉树的直径</font>](https://leetcode.cn/problems/diameter-of-binary-tree/)[简单]
**题目：**给你一棵二叉树的根节点，返回该树的 **直径** 。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点 `root` 。

两节点之间路径的 **长度** 由它们之间边数表示。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764172789967-45c8d586-816c-4f7c-9fb7-d8d52473c4df.png)

**思路：**首先，每一条直径都会有一个顶点(即处于更上层的节点)，那么以某个节点为顶点的最长直径怎么求？

可以等价于该节点的  左子树最大深度+右子树最大深度

以示例一举例，显然最长直径是 4-2-1-3  顶点是1，那么

+ 1的左子树的最大深度是2
+ 1的右子树的最大深度是1
+ 因此以1为顶点的最长直径 = 2+1 = 3

因此分解问题: 求最长直径, 相当于对每一个节点求出左右子树的最大深度之和后，再从中取出去最大值

```java
class Solution {
    private int ans = 0;
    public int diameterOfBinaryTree(TreeNode root) {
        maxDep(root);
        return ans;
    }

    public int maxDep(TreeNode root){
        if(root==null) return 0;
        int a = maxDep(root.left);
        int b = maxDep(root.right);
        ans = Math.max(a+b,ans);
        return Math.max(a,b)+1;
    }
}
```

## [<font style="color:#DF2A3F;">102. 二叉树的层序遍历</font>](https://leetcode.cn/problems/binary-tree-level-order-traversal/)[中等]
**题目：**给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764174767825-899b05c9-1710-41a8-90ec-aa9a90565010.png)

**思路：**使用队列辅助实现，注意初始化时： 把根节点加入队列

当队列不为空的情况下，获取当前队列中的数据个数，意味着当前这一层有几个节点需要弹出

当从队列中取出节点时，要把该节点的左孩子和右孩子以此加入队列，等待下一次循环时再弹出

达到一层一层遍历的目的

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        Deque<TreeNode> dq = new ArrayDeque<>();
        if(root!=null) dq.offerLast(root);

        List<List<Integer>> ans = new ArrayList<>();


        while(!dq.isEmpty()){
            int n = dq.size();
            ArrayList<Integer> arr = new ArrayList<>();

            for(int i = 0;i<n;i++){

                TreeNode node = dq.pollFirst();
                arr.add(node.val);
                if(node.left!=null){
                    dq.offerLast(node.left);
                }
                if(node.right!=null){
                    dq.offerLast(node.right);
                }

            }

            ans.add(arr);
        }
        return ans;

    }
}
```

+ 把 nums[2] 右边的 [5,9] 转换成一棵平衡二叉搜索树，作为答案的右儿子。这是一个和原问题相似的子问题，可以递归解决

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        if(nums.length==0) return null;
        int mid = nums.length/2;
        TreeNode node = new TreeNode(nums[mid]);
        int[] nums1 = Arrays.copyOfRange(nums,0,mid);
        int[] nums2 = Arrays.copyOfRange(nums,mid+1,nums.length);
        node.left = sortedArrayToBST(nums1);
        node.right = sortedArrayToBST(nums2);

        return node;

        
    }
}
```

## [<font style="color:#DF2A3F;">98. 验证二叉搜索树</font>](https://leetcode.cn/problems/validate-binary-search-tree/)[中等]
**题目：**给你一个二叉树的根节点 `root` ，判断其是否是一个有效的二叉搜索树。

**有效** 二叉搜索树定义如下：

+ 节点的左子树只包含** 小于 **当前节点的数。
+ 节点的右子树只包含 **大于** 当前节点的数。
+ 所有左子树和右子树自身必须也是二叉搜索树。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764476593011-6f05b132-7378-4be2-9dfb-2d2806c98e0f.png)

**思路：**

```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        return validate(root, Long.MIN_VALUE, Long.MAX_VALUE);
    }
    
    private boolean validate(TreeNode node, long lower, long upper) {
        if(node == null) return true;
        
        if(node.val <= lower || node.val >= upper) {
            return false;
        }
        
        // 左子树：所有节点必须 < 当前节点，所以上界是 node.val
        // 右子树：所有节点必须 > 当前节点，所以下界是 node.val
        return validate(node.left, lower, node.val) && 
               validate(node.right, node.val, upper);
    }
}
```

## [<font style="color:#DF2A3F;">230. 二叉搜索树中第 K 小的元素</font>](https://leetcode.cn/problems/kth-smallest-element-in-a-bst/)[中等]
**题目：**给定一个二叉搜索树的根节点 `root` ，和一个整数 `k` ，请你设计一个算法查找其中第 `k`小的元素（从 1 开始计数）。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764486695015-426d50f5-6061-49e8-a0d1-c8126140ce85.png)

**思路**：因为是二叉搜索树，所以它的中序遍历结果一定是递增的，因此先求出中序遍历结果，再取出其第k小的数

```java
class Solution {
    // 存储中序遍历结果的列表
    private List<Integer> inorderList;

    public int kthSmallest(TreeNode root, int k) {
        inorderList = new ArrayList<>();
        // 通过中序遍历获取所有节点值
        performInorderTraversal(root);
        // 第k小的数即中序遍历结果中的第k-1个数(注意下标从0开始)
        return inorderList.get(k - 1);
    }

    // 中序遍历
    private void performInorderTraversal(TreeNode node) {
        if (node == null)
            return;

        performInorderTraversal(node.left);
        inorderList.add(node.val);
        performInorderTraversal(node.right);
    }
}
```



## [<font style="color:#DF2A3F;">199. 二叉树的右视图</font>](https://leetcode.cn/problems/binary-tree-right-side-view/)[中等]
**题目：**给定一个二叉树的 **根节点**`root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764487113234-ccbe3280-d65f-4fbe-8e40-147e7581431a.png)

**思路：**层序遍历，遍历每一层时都会有最后一个节点，把每一层的最后一个节点记录下来，即为答案

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        
        // 使用队列(先进先出)辅助 
        Queue<TreeNode> queue = new ArrayDeque<>();
        if (root == null) {
            return res;
        }
        queue.add(root);
        while (!queue.isEmpty()) {
            int size = queue.size(); // 获取这一层数据的个数,然后才知道这一层需要弹出几个元素
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll(); // 弹出根节点
                if (node.left != null) { // 根节点的左子节点加入
                    queue.add(node.left);
                }
                if (node.right != null) { // 根节点的右子节点加入
                    queue.add(node.right);
                }
                if (i == (size - 1)) {
                    res.add(node.val);
                }
            }
        }
        return res;
    }
}
```

## [<font style="color:#DF2A3F;">114. 二叉树展开为链表</font>](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/)[中等]
**题目：**给你二叉树的根结点 `root` ，请你将它展开为一个单链表：

+ 展开后的单链表应该同样使用 `TreeNode` ，其中 `right` 子指针指向链表中下一个结点，而左子指针始终为 `null` 。
+ 展开后的单链表应该与二叉树 [**先序遍历**](https://baike.baidu.com/item/%E5%85%88%E5%BA%8F%E9%81%8D%E5%8E%86/6442839?fr=aladdin) 顺序相同。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764487857602-9fdd8357-1d7c-4378-b3cd-a41662893492.png)

**思路：**

```java
class Solution {
    public void flatten(TreeNode root) {
        if(root == null){
            return;
        }
        flatten(root.left);     //展开左子树
        flatten(root.right);    //展开右子树
        // 暂存右子树
        TreeNode tempRight = root.right;

        // 把展开后的左子树放到右子树的位置上
        root.right = root.left;

        // 把左子树置为空
        root.left = null;
        
        // 先找到展开后的左子树的最右边的节点
        TreeNode curRightest = root;
        while(curRightest.right != null){
            curRightest = curRightest.right;
        }
        // 把之前展开的右子树放在最右边节点的右子树上
        curRightest.right = tempRight;
    }
}
```

## [<font style="color:#DF2A3F;">105. 从前序与中序遍历序列构造二叉树</font>](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)[中等]
**题目：**给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764489862561-a42f78a7-5ed2-426a-b9df-1de2728fb4a2.png)

**思路：**已知前序遍历和中序遍历的结果，那么从这两个数组，我们可以获得什么信息，以示例一举例

+ 从前序遍历的结果数组来看，第一个元素 3 一定是根节点
+ 在中序遍历的结果数组中找到3(由于在题目补充条件中已说明二叉树中的值是各不相同的)，那么3的左边[9]即是左子树，3的右边[15,20,7]即是右子树
+ 通过知道3的左右子树的元素个数，进一步也可以知道在前序遍历中，[9]是左子树，[20,15,7]是右子树

递归，分解问题，构建左子树和右子树的过程都可以通过上述两个步骤进行

例如右子树的在前序遍历中的内容是 [20,15,7]； 在中序遍历中的内容是 [15,20,7]，那么即可知道

+ 右子树的根节点是20，右子树的左子树部分是[15]，右子树的右子树部分是[7]

```java
class Solution {
    
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        int n = preorder.length;
        if(n==0) return null;
        // 在中序里是第k个数
        int k = site(preorder[0],inorder);

        TreeNode root = new TreeNode(preorder[0]);
        

        int[] LPre = Arrays.copyOfRange(preorder,1,k+1);// 1~k
        int[] LIn = Arrays.copyOfRange(inorder,0,k);// 0~k-1
        int[] RPre = Arrays.copyOfRange(preorder,k+1,n);
        int[] RIn = Arrays.copyOfRange(inorder,k+1,n);

        root.left = buildTree(LPre,LIn);
        root.right = buildTree(RPre,RIn);

        return root;
    }


    public int site(int target,int[] inorder){
        for(int i = 0;i<inorder.length;i++){
            if(target==inorder[i]){
                return i;
            }
        }
        return 0;
    }
}
```

## [<font style="color:#DF2A3F;">437. 路径总和 III</font>](https://leetcode.cn/problems/path-sum-iii/)[中等]
**题目：**给定一个二叉树的根节点 `root` ，和一个整数 `targetSum` ，求该二叉树里节点值之和等于 `targetSum` 的 **路径** 的数目。

**路径** 不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的（只能从父节点到子节点）。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764519823980-9e5ba28b-2107-4029-86fd-b1767ab5826e.png)

**思路：**

```java
class Solution {
    public int pathSum(TreeNode root, int targetSum) {
        // 记录前缀和出现了几次,前缀和数组
        Map<Long, Integer> prefixSum = new HashMap<>();
        prefixSum.put(0L, 1);
        return dfs(root, 0, targetSum, prefixSum);
    }
    
    private int dfs(TreeNode node, long currSum, int target, Map<Long, Integer> prefixSum) {
        if (node == null) return 0;
        
        currSum += node.val;
        int res = prefixSum.getOrDefault(currSum - target, 0);
        
        prefixSum.put(currSum, prefixSum.getOrDefault(currSum, 0) + 1);
        res += dfs(node.left, currSum, target, prefixSum);
        res += dfs(node.right, currSum, target, prefixSum);
        prefixSum.put(currSum, prefixSum.get(currSum) - 1);
        

        // 退回来的时候就游戏结束了
        return res;
    }
}
```

## [<font style="color:#DF2A3F;">236. 二叉树的最近公共祖先</font>](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/)[中等]
**题目：**给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

[百度百科](https://baike.baidu.com/item/%E6%9C%80%E8%BF%91%E5%85%AC%E5%85%B1%E7%A5%96%E5%85%88/8918834?fr=aladdin)中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（**一个节点也可以是它自己的祖先**）。”

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764595775984-64536254-3b77-4d95-a72f-6dc07de55d25.png)

**思路**：以示例一的二叉树举例要找到6和4的最近公共祖先，首先

+ 用dfs找到从根节点到6的路径：<font style="color:#DF2A3F;">3 5</font> 6
+ 用dfs找到从根节点到4的路径：<font style="color:#DF2A3F;">3 5</font> 2 4

将这两条路径，分别保存到集合中，然后同时遍历两条路径，找到最后一个相同的节点5，即是最近公共祖先

```java
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        
        if (root == null || root == p || root == q) {
            return root; // 找到 p 或 q 就不往下递归了
        }
        
        TreeNode left = lowestCommonAncestor(root.left, p, q);
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        if (left != null && right != null) { // 左右都找到
            return root; // 当前节点是最近公共祖先
        }
        // 如果只有左子树找到，就返回左子树的返回值
        // 如果只有右子树找到，就返回右子树的返回值
        // 如果左右子树都没有找到，就返回 null（注意此时 right = null）
        return left != null ? left : right;
    }
}
```

## [<font style="color:#DF2A3F;">124. 二叉树中的最大路径和</font>](https://leetcode.cn/problems/binary-tree-maximum-path-sum/)[困难]
**题目：**二叉树中的** 路径** 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 **至多出现一次** 。该路径** 至少包含一个 **节点，且不一定经过根节点。

**路径和** 是路径中各节点值的总和。

给你一个二叉树的根节点 `root` ，返回其 **最大路径和** 。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764595155504-a90ec997-7ffe-4ad3-a109-7286fe0fed51.png)

**思路：**对每个节点，计算“以它为最高点的最大路径”，在过程中记录全局最大值。

我们把大问题分解成小问题：**对于每个节点，计算“以它为转折点的最大路径和”**

什么是"转折点"？比如路径 2-1-3，1 就是转折点，路径从左子树来到右子树。

**关键概念**

1. **单边最大路径**：从某个节点**往下走**（只能选左或右一支）能获得的最大和
2. **以当前节点为转折点的最大路径**：左最大单边路径 + 节点值 + 右最大单边路径

**算法步骤**

1. **从叶子节点开始计算**（递归到底部）
2. **对每个节点**：
    - 计算左子树的“单边最大路径”（如果是负数就取0，因为可以不要）
    - 计算右子树的“单边最大路径”
    - **<font style="color:rgb(15, 17, 21);">更新全局最大值</font>**<font style="color:rgb(15, 17, 21);">：</font>
    `最大路径和 = max(当前最大值, 左+右+当前节点值)`
    - **<font style="color:rgb(15, 17, 21);">返回给父节点 </font>**<font style="color:rgb(15, 17, 21);">为什么返回的是单边路径？：</font>  
    `当前节点的单边最大路径 = 当前节点值 + max(左单边, 右单边)`因为递归时，当前节点需要告诉父节点：“从我这里往上，最好的情况是多少”。父节点只能选择左或右一支，所以返回最大的单边

```java
class Solution {
    int res = Integer.MIN_VALUE;
    public int maxPathSum(TreeNode root) {
        dfs(root);
        return res;
    }
    
    // 这个dfs求的是单边，对吧
    int dfs(TreeNode node){
        if(node == null){
            return 0;
        }
        // 递归求解左子树和右子树的最大单边路径和,如果出现负数的情况则取0
        int leftMax = Math.max(0, dfs(node.left));
        int rightMax = Math.max(0, dfs(node.right));
        // 在递归过程中，计算当前节点为中心向下能找到的最大路径和
        res = Math.max(res, leftMax + rightMax + node.val);
        // 在递归过程中返回的不是“当前节点为中心的最大路径”，而是“从当前节点向上延申能组成的最大单边路径和”，以供父节点使用
        return node.val + Math.max(leftMax, rightMax);
    }
}
```
