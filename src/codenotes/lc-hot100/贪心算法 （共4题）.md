## [<font style="color:#DF2A3F;">121. 买卖股票的最佳时机</font>](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)[简单]
**题目：**给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764609947841-0f8deee0-88a5-42d1-a44a-771c28c97fb9.png)

**思路：**

```java
class Solution {
    public int maxProfit(int[] prices) {
        // 实时计算每次 到当天为止 的最小股票价格和最大利润。
        int cost = Integer.MAX_VALUE, profit = 0;
        for (int price : prices) {
            cost = Math.min(cost, price);
            profit = Math.max(profit, price - cost);
        }
        return profit;
    }
}
```

## [<font style="color:#DF2A3F;">55. 跳跃游戏</font>](https://leetcode.cn/problems/jump-game/)[中等]
**题目：**给你一个非负整数数组 `nums` ，你最初位于数组的 **第一个下标** 。数组中的每个元素代表你在该位置可以跳跃的最大长度。判断你是否能够到达最后一个下标，如果可以，返回 `true` ；否则，返回 `false` 。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764609928542-63dd73b4-5ff9-47ea-8782-b7f8923d3a14.png)

**思路：**<font style="color:rgb(15, 17, 21);">这个问题可以用一种"从后往前"的思路来解决：</font>

1. **<font style="color:rgb(15, 17, 21);">如果我们知道某个位置能到达终点，那么这个位置就成了新的"终点"</font>**
    - <font style="color:rgb(15, 17, 21);">比如最后一个位置本身就是终点</font>
    - <font style="color:rgb(15, 17, 21);">如果倒数第二个位置能跳到最后一个位置，那么问题就变成了：能否到达倒数第二个位置？</font>
2. **<font style="color:rgb(15, 17, 21);">逐步向前推进</font>**
    - <font style="color:rgb(15, 17, 21);">我们从倒数第二个位置开始，向前检查</font>
    - <font style="color:rgb(15, 17, 21);">如果某个位置能跳到当前的"目标位置"，就更新目标位置为这个位置</font>
    - <font style="color:rgb(15, 17, 21);">这样不断向前移动目标位置</font>

最终如果目标位置变成了0，那么说明从位置0到最后一个位置是可达的

```java
class Solution {
    public boolean canJump(int[] nums) {
        // 初始化目标位置是数组的最后一个元素的下标
        int targetIndex = nums.length - 1;
        // 从倒数第二个元素开始逆序遍历数组
        for (int i = nums.length - 2; i >= 0; i--) {
            // 如果从当前位置到目标位置是可达的,那么更新目标位置为当前位置
            if (i + nums[i] >= targetIndex) {
                targetIndex = i;
            }
        }
        // 遍历完数组后，如果 目标 位置 为0，说明从位置0到最后一个位置是可达的
        if (targetIndex == 0) {
            return true;
        }
        return false;
    }
}
```

## [<font style="color:#DF2A3F;">45. 跳跃游戏 II</font>](https://leetcode.cn/problems/jump-game-ii/)[中等]
**题目：**给定一个长度为 `n` 的 **0 索引**整数数组 `nums`。初始位置为 `nums[0]`。

每个元素 `nums[i]` 表示从索引 `i` 向前跳转的最大长度。换句话说，如果你在 `nums[i]` 处，你可以跳转到任意 `nums[i + j]` 处:

+ `0 <= j <= nums[i]`
+ `i + j < n`

返回到达 `nums[n - 1]` 的最小跳跃次数。生成的测试用例可以到达 `nums[n - 1]`。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764689927247-448902ee-9bd4-4e7c-aeea-a8becad84a1f.png)

**思路：**贪心算法，通过局部最优解得到全局最优解，

+ 如果我们「贪心」地进行正向查找，即假设站在下标 i 位置时，可以跳 [0-num[i]]  步远，但是跳几步呢，我们应该要找到下一次能让你跳的更远的位置
+ 例如，对于数组 [2,3,1,2,4,2,3]，初始位置是下标 0，从下标 0 出发，最远可到达下标 2。下标 0 可到达的位置中，下标 1 的值是 3，从下标 1 出发可以达到更远的位置，因此第一步到达下标 1。
+ 从下标 1 出发，最远可到达下标 4。下标 1 可到达的位置中，下标 4 的值是 4 ，从下标 4 出发可以达到更远的位置，因此第二步到达下标 4。
+ 在遍历数组时，只需要遍历到倒数第二个位置，最后一个位置是不需要再跳的

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/54710028/1764692006097-86495e17-634e-4173-a85c-b408c442118f.png)

**可参考思路讲解：**[45. 跳跃游戏 II](https://www.bilibili.com/video/BV1Y24y1r7XZ?spm_id_from=333.788.videopod.sections&vd_source=9297c4d03d5f37a56605a3cb4627b688)

具体实现：

+ `**<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">curRangeEnd</font>**`<font style="color:rgb(15, 17, 21);">：当前跳跃轮次的"访问权限边界"</font>
    - <font style="color:rgb(15, 17, 21);">在上一跳的影响下，这一轮允许探索到哪里</font>
    - <font style="color:rgb(15, 17, 21);">初始化0：第0轮只允许探索位置0</font>
+ `**<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">nextRangeEnd</font>**`<font style="color:rgb(15, 17, 21);">：下一跳跃轮次的"能力边界"</font>
    - <font style="color:rgb(15, 17, 21);">在当前轮次遍历过程中，收集下一轮能跳到的最远处，随着遍历而更新</font>
+ `**<font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">jumps</font>**`<font style="color:rgb(15, 17, 21);">：已执行的跳跃动作次数</font>
    - <font style="color:rgb(15, 17, 21);">从位置0到位置0：0次跳跃（起点）</font>
    - <font style="color:rgb(15, 17, 21);">从位置0跳到位置1：1次跳跃</font>

```java
class Solution {
    public int jump(int[] nums) {
        int curRangeEnd = 0;
        int nextRangeEnd = 0;
        int jumps = 0;
        
        for (int i = 0; i < nums.length - 1; i++) {
            // 更新最远边界
            nextRangeEnd = Math.max(nextRangeEnd, i + nums[i]);
            
            if (i == curRangeEnd) {
                curRangeEnd = nextRangeEnd;
                jumps++;
            }
        }
        
        return jumps;
    }
}
```

## [<font style="color:#DF2A3F;">763. 划分字母区间</font>](https://leetcode.cn/problems/partition-labels/)[中等]
**题目：**给你一个字符串 `s` 。我们要把这个字符串划分为尽可能多的片段，<font style="color:#DF2A3F;">同一字母最多出现在一个片段中</font>。

注意，划分结果需要满足：将所有划分结果按顺序连接，得到的字符串仍然是 `s` 。

返回一个表示每个字符串片段的长度的列表。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1764696070073-8e5a1a4a-2c41-45eb-9261-4a62463a9760.png)

**思路：**由于同一个字母只能出现在同一个片段，显然同一个字母的第一次出现的下标位置和最后一次出现的下标位置必须出现在同一个片段。

1. **<font style="color:rgb(15, 17, 21);">记录每个字母最后出现的位置：</font>**<font style="color:rgb(15, 17, 21);">扫描一遍字符串，记录每个字母最后一次出现的位置</font>
2. **<font style="color:rgb(15, 17, 21);">开始划分片段，动态扩展当前片段（start、end）的边界end</font>**
    - <font style="color:rgb(15, 17, 21);">从左到右扫描字符串</font>
    - <font style="color:rgb(15, 17, 21);">对于每个字符，查看它的最后出现位置</font>
    - <font style="color:rgb(15, 17, 21);">不断扩展当前片段的结束位置</font>
    - <font style="color:rgb(15, 17, 21);">当走到当前片段的边界时，就完成了一个片段，</font>
3. <font style="color:rgb(15, 17, 21);">置start=end+1后，继续寻找下一个片段</font>

```java
class Solution {
    public List<Integer> partitionLabels(String s) {
        int[] last = new int[26];
        int length = s.length();
        // 通过这种方式遍历一遍，有重复字母会以覆盖的方式更新下标，
        // 自然就记录了每个字母最后出现的位置
        for (int i = 0; i < length; i++) {
            last[s.charAt(i) - 'a'] = i;
        }
        
        List<Integer> ans = new ArrayList<>();
        int start = 0;
        int end = 0;
        
        for (int i = 0; i < length; i++) {
            // end可能会有连续扩大的情况，如 a.......b......a.....b
            // 还没走到第二个a呢，end就扩大到第二个b了
            // 这就巧妙实现了区间的更新
            end = Math.max(end, last[s.charAt(i) - 'a']);
            if (i == end) {
                ans.add(end - start + 1);
                start = end + 1;
            }
        }
        
        return ans;
    }
}
```
