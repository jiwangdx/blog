## [<font style="color:#DF2A3F;">283. 移动零</font>](https://leetcode.cn/problems/move-zeroes/)[简单]
给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。**请注意** ，必须在不复制数组的情况下原地对数组进行操作。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1751610251371-03f0d79c-53d2-4d4b-b028-c8232d7b94c8.png)

**思路：**

**两次遍历**

+ 第一次遍历：将所有非零元素移到左侧
    - 定义两个指针，`<font style="background-color:rgba(100, 100, 100, 0.08);">slow = 0, fast = 0</font>`，<font style="color:#DF2A3F;">不断移动fast指针</font>，在循环内，<font style="color:#DF2A3F;">如果</font>fast指针指向元素不是零，则把该元素放到slow指针对应的位置，并且<font style="color:#DF2A3F;">slow指针右移</font>一位
+ 第二次遍历：剩下的元素都置为0

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int n = nums.length;
        int l = 0;
        for(int r = 0;r<n;r++){
            if(nums[r]!=0){
                nums[l] = nums[r];
                l++;
            }
        }
        // l位置之后的都是0
        for(int i = l;i<n;i++){
            nums[i] = 0;
        }
    }
}
```

## [<font style="color:#DF2A3F;">11. 盛最多水的容器</font>](https://leetcode.cn/problems/container-with-most-water/)[中等]
给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1751614237730-b8cc8186-7245-4cc4-84e8-ec86a4c92e97.png)

**思路：**

题意为：求如上图围成的最大面积，面积等于宽×高  `宽为 j-i` `高为 min(height[i], height[j])`

即给定数组`height`，任选两个下标`i`和`j`，求`(j-i) * min(height[i],height[j])`的最大值

双指针，左指针从最左侧开始向右检索，右指针从最右侧开始向左检索，

每次只移动高度较小的那个指针，遍历过程中计算出当前围成的面积，并且记录最大面积

```java
class Solution {
    public int maxArea(int[] height) {
        int n = height.length;
        int l = 0,r = n-1;

        int ans = 0;
        while(l<r){
            ans = Math.max(ans,(r-l)*Math.min(height[l],height[r]));
            if(height[l]<=height[r]){
                l++;
            }
            else{
                r--;
            }
        }
        return ans;
    }
}
```

## [<font style="color:#DF2A3F;">15. 三数之和</font>](https://leetcode.cn/problems/3sum/)[中等]
给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1752983959851-43dd4e0e-f459-46f3-83a9-51e3a25e0dd5.png)

**思路：**

+ 固定一个数，将三数之和转换为两数之和
+ 数组排序，使用双指针求解两数之和
+ 移动指针过程中，对组合去重

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> ans = new ArrayList<>();
        List<Integer> cache = new ArrayList<>();
        int n = nums.length;
        Arrays.sort(nums);

        for(int i = 0;i<=n-3;i++){
            if(nums[i]>0) break;
            // 这里的状态是：当前循环的业务逻辑还没开始，上一轮的已经结束，所以跟上一轮去比
            if(i>=1 && nums[i]==nums[i-1]) continue;
            int l = i+1;
            int r = n-1;
            // 用while循环更好理解
            while(l<r){
                if(nums[i]+nums[l]+nums[r]==0){
                    ans.add(List.of(nums[i],nums[l],nums[r]));
                    while(l<r && nums[r]==nums[r-1]){
                        r--;
                    }
                    while(l<r && nums[l]==nums[l+1]){
                        l++;
                    }
                    r--;
                    l++;
                }
                else if(nums[i]+nums[l]+nums[r]>0){
                    r--;
                }
                else{
                    l++;
                }
            }
        }

        return ans;


    }
}
```

## [<font style="color:#DF2A3F;">42. 接雨水</font>](https://leetcode.cn/problems/trapping-rain-water/)[困难]
给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水

<!-- 这是一张图片，ocr 内容为： -->
![](https://jiwang.cc.cd/PicGo/1752983987704-703695a1-05ba-49c9-9947-12da46dccada.png)

**思路：**

每个柱子可接的雨水的高度 = `Min（左边区间的最高柱子，右边区间的最高柱子）- 当前柱子的高度`

```java
class Solution {
    public int trap(int[] nums) {
        int n = nums.length;
        // 当前格子的雨水=左右两边最大柱子中的较小者-当前柱子，如果小于等于0,那就没水。
        int[] lmax = new int[n];// 包含当前
        int[] rmax = new int[n];
        lmax[0] = nums[0];
        for(int i = 1;i<n;i++){
            lmax[i] = Math.max(lmax[i-1],nums[i]);
        }
        rmax[n-1] = nums[n-1];
        for(int i = n-2;i>=0;i--){
            rmax[i] = Math.max(rmax[i+1],nums[i]);
        }

        int ans = 0;
        for(int i =0;i<n;i++){
            if(Math.max(lmax[i],rmax[i])-nums[i]>0) ans+=Math.min(lmax[i],rmax[i])-nums[i];
        }

        return ans;

    }
}
```
