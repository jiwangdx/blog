05 普通数组

jiwang3/4/26About 8 min代码笔记LC Hot100普通数组

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。子数组是数组中的一个连续部分。

思路：

dp[i]表示以nums[i]为结尾的最大子数组和

class Solution {
    public int maxSubArray(int[] nums) {
        int n = nums.length;
        int[] dp = new int[n];
        dp[0] = nums[0];
        int ans = nums[0];
        for(int i = 1;i<n;i++){
            dp[i] = Math.max(dp[i-1]+nums[i],nums[i]);
            ans = Math.max(ans,dp[i]);
        }

        return ans;
    }
}

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [start(i), end(i)] 。请你合并所有重叠的区间，并返回 一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间 。


思路：

输入：[[1,3],[2,6],[8,10],[15,18]]
处理过程：
步骤1：排序后 → [[1,3],[2,6],[8,10],[15,18]]
步骤2：当前区间 = [1,3]
步骤3：处理[2,6] → 有重叠 → 当前区间变为[1,6]
步骤4：处理[8,10] → 无重叠 → 保存[1,6]，新当前区间[8,10]
步骤5：处理[15,18] → 无重叠 → 保存[8,10]，新当前区间[15,18]
步骤6：保存最后区间[15,18]
结果：[[1,6],[8,10],[15,18]]
  1. 先按照 区间的左端点进行排序：
情况1：有重叠（如[1,3]和[2,6]）
● 判断条件： 新区间起点 <= 当前合并区间终点
● 处理方式： 扩展当前合并区间终点（取最大值）
● 示例：
  ○ 当前合并区间：[1,3]
  ○ 新区间：[2,6] → 终点取max(3,6)=6
  ○ 合并后：[1,6]
情况2：无重叠（如[1,6]和[8,10]）
● 判断条件： 新区间起点 > 当前合并区间终点
● 处理方式：
  a. 保存当前合并区间到结果列表
  b. 将新区间设为新的当前合并区间
● 示例：
  ○ 保存[1,6]到结果
  ○ 新当前区间设为[8,10]
  1. 收尾工作：
class Solution {
    public int[][] merge(int[][] intervals) {
        // 合并所有重叠的区间
        List<int[]> ans = new ArrayList<>();
        Arrays.sort(intervals, (a, b) -> a[0] - b[0]);// 按照第一个区间点进行排序
        // 什么是重叠，前面的end超过了后面的start
        int n = intervals.length; //一共有n组点对

        int lastStart = intervals[0][0];
        int lastEnd = intervals[0][1];
        int curStart = 0;
        int curEnd = 0;
        for (int i = 1; i < n; i++) {

            curStart = intervals[i][0];
            curEnd = intervals[i][1];

            if (lastEnd >= curStart) {
                // 新点对的尾巴取决于长的那个
                lastEnd = Math.max(lastEnd, curEnd);
            } else {
                ans.add(new int[]{lastStart,lastEnd});
                lastStart = curStart;
                lastEnd = curEnd;
            }
        }
        ans.add(new int[]{lastStart,lastEnd});

        // return ans.toArray(new int[0][]);
        return ans.toArray(new int[ans.size()][]);

    }
}

给定一个整数数组 nums，将数组中的元素向右轮转 k个位置，其中 k是非负数。

思路：

首先我们先对**<font style="background-color:rgba(100, 100, 100, 0.08);">nums</font>**进行一次反转，反转后为<font style="background-color:rgba(100, 100, 100, 0.08);">[7,6,5,4,3,2,1]</font>，观察下反转以后的元素的分布。

我们只需要对反转后的数组中前**<font style="background-color:rgba(100, 100, 100, 0.08);">k</font>**个元素和后**<font style="background-color:rgba(100, 100, 100, 0.08);">n - k</font>**个元素再分别反转一次就可以得到轮转后的结果

class Solution {
    public void rotate(int[] nums, int k) {
        if(nums.length<=1) return;
        // 三次翻转
        int n = nums.length;
        k = k%n;
        reverse(nums,0,n);
        reverse(nums,0,k);
        reverse(nums,k,n);
    }
    public void reverse(int[] nums,int start,int end){
        
        while(start<end){
            int temp = nums[start];
            nums[start] = nums[end-1];
            nums[end-1] = temp;
            start++;
            end--;
        }
    }
}

给你一个整数数组 nums，返回 数组 answer ，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积 。

题目数据 保证 数组 nums之中任意元素的全部前缀元素和后缀的乘积都在 32 位 整数范围内。

请 **不要使用除法，**且在 O(n) 时间复杂度内完成此题。

思路：

结果 res[i] 等于 nums 中除了 nums[i] 之外其余各元素的乘积

换句话说，如果知道了 i 左边所有数的乘积，以及 i 右边所有数的乘积，就可以算出 结果 res[i]

例如 如果要计算除了3之外的乘积。可以认为是 3左边所有数的乘积 * 3右边所有数的乘积

于是引入两个概念，前缀积和后缀积

  • 前缀积：第i个元素的前缀积为第i个元素之前所有元素的乘积 pre[i] = pre[i - 1] * nums[i - 1];
  • 后缀积：第i个元素的后缀积为第i个元素之后所有元素的乘积 suf[i] = suf[i + 1] * nums[i + 1];

那么结果 res[i] = 前缀积pre[i] * 后缀积suf[i]

因此只需要先遍历两次计算出前缀积和后缀积，再求得结果数组即可

class Solution {
    public int[] productExceptSelf(int[] nums) {
        // 答案等于前缀乘积乘以后缀乘积
        int n = nums.length;
        int[] mul_f = new int[n];
        int[] mul_b = new int[n];
        int[] answer = new int[n];
        mul_f[0] = 1;
        mul_f[1] = nums[0];
        for (int i = 2; i < n; i++) {
            mul_f[i] = mul_f[i - 1] * nums[i - 1];
        }
        mul_b[n - 1] = 1;
        mul_b[n - 2] = nums[n - 1];
        for (int i = n-3; i >= 0; i--) {
            mul_b[i] = mul_b[i + 1] * nums[i + 1];
        }

        for(int i = 0;i<n;i++){
            answer[i] = mul_b[i]*mul_f[i];
        }

        return answer;

    }
}

给你一个未排序的整数数组 nums ，请你找出其中没有出现的最小的正整数。

请你实现时间复杂度为 O(n) 并且只使用常数级别额外空间的解决方案。

思路：

  • **<font style="color:rgb(64, 64, 64);background-color:rgb(236, 236, 236);">[1, n+1]</font>****<font style="color:rgb(64, 64, 64);background-color:rgb(236, 236, 236);">n</font>**

class Solution {
    public int firstMissingPositive(int[] nums) {
        int n = nums.length;
        for(int i = 0;i<n;i++){
            while(0<nums[i]&&nums[i]<n+1&&nums[nums[i]-1]!=nums[i]){ 
                //一次交换过后又把一个新的脏东西拿上来了，
                // 这个脏东西不一定恰好放在了它该放的位置，
                // 因此可能需要多次处理,采用while循环最为合适
                swap(nums,nums[i]-1,i);
            }
        }

        for(int i = 0;i<n;i++){
            if(nums[i]!=i+1) return i+1;
        }

        return n+1;
            
    }

    public void swap(int[] nums,int i,int j){
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
