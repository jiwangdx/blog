03 滑动窗口

jiwang3/4/26About 2 min代码笔记LC Hot100滑动窗口

[中等]

给定一个字符串 s ，请你找出其中不含有重复字符的 **最长子串 **的长度。

思路：

由两个索引维护一个区间，满足一定条件时会动态的去调整这个区间。

class Solution {
    public int lengthOfLongestSubstring(String s) {
        // 字符串的比较
        if(s.equals("")) return 0;

        // 有一个窗口一直用来存无重复字符的子串
        // 整个队列吧
        int n = s.length();
        Deque<Character> window = new ArrayDeque<>();

        int ans = 1;
        for(int i = 0;i<n;i++){
            
            while(window.contains(s.charAt(i))){
                // poll()方法默认是队头出列
                window.poll();
            };
            window.offer(s.charAt(i));

            // 这时窗口里边都是满足要求的
            ans = Math.max(ans,window.size());      
        }
        return ans;
    }
}

[中等]

给定两个字符串 s 和 p，找到 s中所有 p的 **异位词 **的子串，返回这些子串的起始索引。不考虑答案输出的顺序。


思路：

因为字符串p的长度是固定的，那么就需要在字符串s中找到长度相同的子串

即需要遍历字符串s中所有长度与p相同的子串，此时就可以使用滑动窗口来解决

把当前窗口组成的子串排序，以及字符串p内部也排序，如果两个字符串排序后相同，则说明当前窗口的子串就是字符串p的异位词。

class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        String flag = tool(p);// 找到判断异位词的标杆

        int len_s = s.length();
        int len_p = p.length(); 
        List<Integer> ans = new ArrayList<>();
        for(int i = 0;i<=len_s-len_p;i++){
            String temp = s.substring(i,i+len_p);
            if(tool(temp).equals(flag)) ans.add(i) ;
        }
        return ans;    
    }



    public String tool(String str){
        char[] chs = str.toCharArray();
        Arrays.sort(chs);
        return new String(chs);
    } 
}
